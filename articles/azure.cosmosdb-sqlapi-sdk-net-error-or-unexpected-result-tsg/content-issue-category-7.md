# Issue Category

Customers may face variety of issues related to SDK (SQL API) error or unexpected result such as:

1. .Net: SQL API Get Request Returns Bad Request (401 Errors)
2.  403 Forbidden
3. 429 Too Many Requests
4. Date header doesn't conform to the required format
5. Invalid characters in Resource.Id _rid Property
6. The read session is not available for the input session token
7. Intermittent 503 errors service is unavailable
8. Unable to connect to Cosmos DB Account from Java SDK (Async API) - Using SQL API
9. IdleEndpointTimeout message from some connections Cosmos DB
10. Stored procedure cannot insert due to partition key mismatch
11. Using UpsertItem
12. Error - Cross partition query is required but disabled
13. Java Sprint Boot SDK fails to connect to Azure Cosmos DB using Direct Connection Mode
14. .Net: WebProxy Binding with Cosmos request
