# Azure VMware Solution

The cloudadmin user that you use to access vCenter is in the CloudAdmin group. That group establishes the privileges and permitted vCenter actions that the cloudadmin user has in a private cloud. Review the documentation for vCenter actions available to the CloudAdmin group.
