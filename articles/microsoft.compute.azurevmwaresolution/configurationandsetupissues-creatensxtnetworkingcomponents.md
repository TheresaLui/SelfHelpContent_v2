# Azure VMware Solution

AVS private clouds are deployed with a fully operational NSX-T Manager that has includes pre-provisioned T0 and T1 gateways. You will create new network segments and attach them to an existing T1 gateway. Other NSX-T components and services can be created using instructions available from VMware.

Note:  Any changes to the pre-provisioned NSX-T T0 or T1 gateways will result in degraded or disruptive operation of your private cloud.
