# Resource Limitations

### Symptom

Customer claims that UI is lagging because of the number of cells or commands in the notebook.

### Troubleshooting & Solution

Check customer notebook, identify the following parametrs, and make sure its not crossing advised limit:

Number of cells per notebook 		~100
Number of lines in each notebook cell 	~1000 lines
Notebook history size (uncompressed)	~100MB
Workspace export max size		~50 MB*
Number of onjects in workspace folder	~1000
