# Diagnose and resolve issues with source control integration – Github 

## **Recommended Steps** 

* Integration with GitHub Enterprise Server is not supported. However, you can use the [Workspace API](https://docs.microsoft.com/azure/databricks/dev-tools/api/latest/workspace) to programmatically create notebooks and manage the code base in GitHub Enterprise Server.
