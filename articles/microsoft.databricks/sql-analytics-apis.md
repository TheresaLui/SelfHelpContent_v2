# Diagnose and resolve issues with SQL Analytics APIs 

* This feature is in **Private Preview**. To try it, reach out to your Azure Databricks contact. 
 

## **Recommended Steps** 

* To access Databricks REST APIs, you must [authenticate](https://docs.microsoft.com/azure/databricks/sql/api/authentication). 
* To **create SQL endpoints** you must have [cluster create permission](https://docs.microsoft.com/azure/databricks/administration-guide/access-control/cluster-acl#cluster-create-permission) in Azure Databricks Workspace. 
* To **manage a SQL endpoint** you must have [Can Manage](https://docs.microsoft.com/azure/databricks/sql/user/security/access-control/sql-endpoint-acl) permission in Azure Databricks SQL Analytics for the endpoint.
