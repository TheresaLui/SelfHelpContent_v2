# Ingestion Data Container cannot be deleted
<!--issueDescription-->
The resource you are trying to delete is the ‘ingestiondata’ container created by Office 365 services. This container is used to upload Outlook PST files to Office 365. Customers do not have the ability to delete data in this container. These containers will be deleted automatically after a period of time. If you need this container deleted prior to the automatic deletion, then please let us know and we can escalate this ticket to the Office 365 team.    
<!--/issueDescription-->
