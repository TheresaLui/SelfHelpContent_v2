# How to check ASC for a Storage account error

1. ASC insights provide information on the error and proper repair steps. Please follow the ASC insight using the below steps
2. Make sure that the case details are correct. Details include
   1. Subscription Id
   2. Resource
   3. Resource Type
   4. Support Topic
3. If details are not correct, click on Edit & Run Again and correct/complete the information. Otherwise, continue.
   1. Select the Insights section
   2. Review the Critical, Warning and Informational Insights displayed and evaluate if the Insights correlate to the issue the customer is facing.
   3. If the Insights are present and related to the issue, please execute the Recommended Actions within it.
