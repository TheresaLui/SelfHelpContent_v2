# Troubleshooting vCenter access 

## **Recommended Steps**

*  Check your email for CloudSimple maintenance notification for this time
*  Check that VPN connection to the Private Cloud is functional
*  Check that routing is configured correctly for the VPN
*  Check that the browser proxy allows connecting to vCenter
*  Check that vCenter Server host name is resolved correctly by your DNS
*  Check that your DNS is configured correctly for forwarding name resolution
*  Shockwave Flash plugin crashes in the browser?
*  Try connecting using a private browser session (aka incognito mode)
