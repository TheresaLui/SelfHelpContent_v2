If you are haveing issues with SQL Server Managemt Studio or Azure Data Studio, please ensure that you are using **[the latest build of SQL Server Managemt Studio](https://docs.microsoft.com/sql/ssms/download-sql-server-management-studio-ssms?view=sql-server-ver15)** or **[the latest build of Azure Data Studio](https://docs.microsoft.com/sql/azure-data-studio/download-azure-data-studio?view=sql-server-ver15)** to avoid any known issues.
