## **Recommended Documents**

The Run Command feature uses the virtual machine (VM) agent to run shell scripts within an Azure Linux VM. You can use these scripts for general machine or application management. They can help you to quickly diagnose and remediate VM access and network issues and get the VM back to a good state.

* [Overview](https://docs.microsoft.com/azure/virtual-machines/linux/run-command)<br>
* [Restrictions](https://docs.microsoft.com/azure/virtual-machines/linux/run-command#restrictionss)<br>
* [Available commands](https://docs.microsoft.com/azure/virtual-machines/linux/run-command#available-commands)<br>
* [Limiting access to Run Command](https://docs.microsoft.com/azure/virtual-machines/linux/run-command#limiting-access-to-run-command)<br>
