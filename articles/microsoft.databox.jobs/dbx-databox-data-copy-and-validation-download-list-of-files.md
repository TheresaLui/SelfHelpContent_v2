# Data Box - Data copy and validation
Ensure that the Data Box has completed prepare to ship without any critical errors. The BOM files or manifest is available for download once this step completes successfully. 
In case of a Data Box Disk order, ensure that the data copied to each folder is in the right format and any errors logged while running the validation tool are fixed.
