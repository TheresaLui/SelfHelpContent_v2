# Can't schedule a pick-up

For Data Box orders, the return shipping label and reference number will be available in order to schedule a pick-up only after completing Prepare to ship without any critical errors. If you have the reference number after successful completion of this step, and are facing issues with scheduling a pickup with the carrier, contact [Azure Data Box Operations](adbops@microsoft.com)

For any other issues with pick-up for Data Box and Data Box Disk orders, check the escalation path provided under Overview > View ship instructions in the Azure portal for your order.
