# Data Box - Data copy and validation

**Copy is slow**

## **Recommended Steps**

Perform the following steps before you reach out to support

1. Take a picture of the damaged package
2. Do not use the device/disks
3. Continue to create support request
4. Send back the device/disks
5. Create a new order for device/disks
