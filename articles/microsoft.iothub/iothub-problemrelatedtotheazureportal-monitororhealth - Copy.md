# IoT Hub logging and metrics

## **Recommended Steps**

1. Learn more about IoT Hub's [logging](https://docs.microsoft.com/azure/iot-hub/iot-hub-monitor-resource-health#understand-the-logs) and [platform metrics](https://docs.microsoft.com/azure/iot-hub/iot-hub-metrics), available through Azure Monitor. <br>
1. Looking to add metric alerts? Go to [Alerts](data-blade:Microsoft_Azure_Monitoring.AlertsManagementSummaryBlade.id.$resourceId), then click "View classic alerts". [Alternatively, click here to go to classic alerts directly](data-blade:Microsoft_Azure_Monitoring.AlertsBlade.id.$resourceId)
