# I can't restart or resize an existing VM

## **Recommended Steps**

To resolve common issues, try one or more of the following methods:

1. Review [Audit logs](data-blade:Microsoft_Azure_Insights.AzureDiagnosticsBladeWithParameter.subscriptionId.$subscriptionId) to determine the failure reason
2. Look up suggested actions by error code and [Troubleshoot error codes for resize & restart](https://docs.azure.cn/virtual-machines/windows/classic/troubleshoot-deployment-new-vm#error-string-lookup)
3. Try your request using a smaller VM size or using a different cloud service by [Configuring regional Virtual Network to connect Cloud services](https://azure.microsoft.com/blog/vnet-to-vnet-connecting-virtual-networks-in-azure-across-different-regions/)
