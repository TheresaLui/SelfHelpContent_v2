**Troubleshooting**


* RBAC new roles Key Vault Administrator, Key Vault Reader and Key Vault Secrets, Certificates and Keys roles are currently in private preview. It require subscription to be enabled for roles to take affect. Please contact jalichwa@microsoft.com for more information
