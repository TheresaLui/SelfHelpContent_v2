# Encountering an Unsupported Command issue

<!--issueDescription-->

Dear customer,

If you are encountering an Unsupported Command:
* Consider Migration to [MongoDB (3.6 version)](https://docs.microsoft.com/azure/cosmos-db/mongodb-feature-support-36)
* Review and enable preview feature via the Azure Portal, for example, Mongo Aggregation Pipeline, or Mongo 3.4 compatibility [v3.2]
* Rewrite query minus the unsupported operator, with client-side logic when required  [v3.6]

Thank you.
