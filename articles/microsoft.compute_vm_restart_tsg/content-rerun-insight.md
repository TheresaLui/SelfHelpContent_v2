# How to check if ASC insight is correct

* Go to the ASC portal and on the Insights overview, make sure everything is accurate for insights
* If the Support Topic is not correct or even the selected resource, select the correct support topic and resource with Edit & Run Again. 
* Once you specify the correct ones, the insight analyzer will be triggered again. 
* In several minutes, you will get the new insights result. 
* After it finishes, please review the insights result again.
* If the information on Insights are accurate, go ahead and check it. 
* They may appear with two types of titles: **VM RESTART DETECTED** or **SERVICE HEALTH**
