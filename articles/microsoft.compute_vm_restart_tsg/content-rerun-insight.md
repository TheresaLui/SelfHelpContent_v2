# How to check if ASC insight is correct

1. Go to the ASC portal and on the Insights overview, make sure everything is accurate for insights
2. If the Support Topic is not correct or even the selected resource is incorrect, select the correct support topic and resource with Edit & Run Again. 
3. Once you specify the correct ones, the insight analyzer will be triggered again. 
4. In several minutes, you will get the new insights result. 
5. After it finishes, please review the insights result again.
6. If the information on Insights are accurate, go ahead and check the insights. 
7. They may appear with two types of titles: **VM RESTART DETECTED** or **SERVICE HEALTH**
