# Solutino for Mac OS does not support Azure AD Authentication

We found that this is not a supported scenario. Mac OS does not support Azure AD Authentication. 

## Recommended Steps

1. Use a Windows Azure VPN client with OpenVPN to leverage Azure AD authentication
