# Solution for VPN Point-to-Site client error 0x800b0109

Thank you for working with us to identify the cause of the issue you are experiencing. This issue was caused by the VPN gateway being updated/upgraded or deleted/recreated. The solution to this issue is to download and install the updated VPN P2S client package again to get the updated certificate.

We believe this has met your need but feel free to reach out for assistance if needed. We are sorry for the inconvenience. If you have thoughts on how we can improve we would appreciate any feedback you can offer.

## Recommended Steps

1. Redownload package from portal and re-install on all clients
