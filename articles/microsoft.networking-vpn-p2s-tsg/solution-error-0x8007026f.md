# Solution for VPN Point-to-Site client error 0x8007026f

The error *"Error 0x8007026f: Azure VPN Custom script (to update your routing table) failed"* happens when the VPN client application is launched from a desktop shortcut. The VPN client does not support launching from a desktop shortcut.

## Recommended Steps

1. Open the VPN client application directly without desktop shortcut
