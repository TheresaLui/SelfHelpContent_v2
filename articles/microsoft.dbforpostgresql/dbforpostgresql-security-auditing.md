# Auditing capabilities in Azure Database for PostgreSQL

## **Recommended Steps**

The Azure Activity Log provides information about subscription-level events. Your Activity Log for Azure Database for PostgreSQL will log ARM-level events like server creation, server scaling, and server delete. Learn more about [Activity Log](https://docs.microsoft.com/azure/azure-monitor/platform/activity-logs-overview). 

An Azure Database for PostgreSQL database-level audit feature is planned but not yet available. If you are interested in this feature you can vote for it on [our feedback page](https://feedback.azure.com/forums/597976-azure-database-for-postgresql/suggestions/20058184-pgaudit-extension). Voting will allow you to get updates on the roadmap of this feature as they are announced.
