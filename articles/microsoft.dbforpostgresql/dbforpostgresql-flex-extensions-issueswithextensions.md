# Issues with PostgreSQL extensions

## **Recommended steps**

* We recommend that you review your specific extension's documentation to learn more about it. The [flexible server extensions document](https://docs.microsoft.com/azure/postgresql/concepts-extensions) provides links to each extension's documentation. 

* Use the command `SHOW azure.extensions;` to see the list of available extensions on your flexible server. You can also review the [flexible server extensions documentation](https://docs.microsoft.com/azure/postgresql/concepts-extensions).
