# Manage firewall rules
You can [create firewall rules](https://docs.microsoft.com/azure/postgresql/howto-hyperscale-manage-firewall-using-portal) to allow traffic from your approved IP addresses to the Hyperscale (Citus) coordinator node.

## **Recommended Steps**
* Only firewall rules are supported for network access control. Virtual networks are not currently a supported feature in Hyperscale (Citus).

## **Recommended Documents**
* [Creating and managing firewall rules](https://docs.microsoft.com/azure/postgresql/howto-hyperscale-manage-firewall-using-portal)
