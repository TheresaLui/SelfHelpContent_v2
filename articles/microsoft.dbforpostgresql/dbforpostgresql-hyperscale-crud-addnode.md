# Issues adding nodes

In Azure Database for PostgreSQL - Hyperscale (Citus) you can add more worker nodes to scale up your cluster.

## **Recommended Steps**
* After you [add worker nodes](https://docs.microsoft.com/azure/postgresql/howto-hyperscale-scaling#add-worker-nodes), remember to [rebalance shards](https://docs.microsoft.com/azure/postgresql/howto-hyperscale-scaling#rebalance-shards) to make use of the new nodes
* To increase the amount of compute and storage allocated to existing worker nodes, create a support request for scaling node compute and storage

## **Recommended Documents**
* [Scaling Hyperscale](https://docs.microsoft.com/azure/postgresql/howto-hyperscale-scaling)
