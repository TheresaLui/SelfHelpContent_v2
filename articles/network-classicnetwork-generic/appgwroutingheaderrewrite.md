# Header Rewrite

Application gateway supports the capability to add, remove, or update HTTP request and response headers while the request and response packets move between the client and backend pools. It provides you the capability to add conditions to ensure that the specified headers are rewritten only when certain conditions are met.

## **Recommended Documents**

* Configure HTTP Header rewrite in Application Gateway using [Azure Portal](https://docs.microsoft.com/en-us/azure/application-gateway/rewrite-http-headers-portal) or [PowerShell](https://docs.microsoft.com/en-us/azure/application-gateway/add-http-header-rewrite-rule-powershell)<br />
* [HTTP Header rewrite with conditions and server variables](https://docs.microsoft.com/en-us/azure/application-gateway/rewrite-http-headers) overview on Application Gateway
