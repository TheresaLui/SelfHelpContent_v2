# connectivity/mypublicipchanged

## **Recommended Steps**

If you've lost your public IP address because your VM was stopped, it cannot be recovered. There is nothing support can do to recover your IP address.

1. [Creating a VM with a static IP address](https://docs.microsoft.com/azure/virtual-network/virtual-network-deploy-static-pip-arm-portal) will prevent change of your Public IP address in the future
2. Learn about the [Public IP addresses types](https://docs.microsoft.com/azure/virtual-network/virtual-network-ip-addresses-overview-arm#public-ip-addresses) in Azure
