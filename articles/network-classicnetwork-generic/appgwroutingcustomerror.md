# Custom Error Page

Application Gateway allows you to create custom error pages instead of displaying default error pages. You can use your own branding and layout using a custom error page.

**Note**: The custom error page URL has to be in an Azure Storage Blob container with *.htm or *.html file type and the blob should be publicly accessible.

## **Recommended Documents**

Configure Custom Error pages in Application Gateway using [Azure Portal and PowerShell](https://docs.microsoft.com/azure/application-gateway/custom-error)
