# Configure ExpressRoute Gateway

You can create a Gateway connection to link a virtual network to an ExpressRoute circuit.<br>

## **Recommended documents**

*	[Configure a Virtual Network Gateway](https://docs.microsoft.com/azure/expressroute/expressroute-howto-add-gateway-portal-resource-manager) for ExpressRoute<br>
*	Connect a Virtual Network to an ExpressRoute circuit using the [portal](https://docs.microsoft.com/azure/expressroute/expressroute-howto-linkvnet-portal-resource-manager), [PowerShell](https://docs.microsoft.com/azure/expressroute/expressroute-howto-linkvnet-arm) or [CLI](https://docs.microsoft.com/azure/expressroute/howto-linkvnet-cli)<br>
*	After the setup is complete, you can check the [ExpressRoute Gateway connections in bits/seconds](https://docs.microsoft.com/azure/expressroute/expressroute-howto-add-gateway-portal-resource-manager)<br>
*	Learn more about [Virtual Network Gateways for ExpressRoute](https://docs.microsoft.com/azure/expressroute/expressroute-about-virtual-network-gateways)<br>
*	[NAT requirements](https://docs.microsoft.com/azure/expressroute/expressroute-nat) for ExpressRoute setup
