# Application Gateway Performance

If you are facing any performance issues such as high response times and latencies while accessing Application Gateway and if you are using the V1 SKU, consider evaluating Application Gateway V2 SKU which provides Autoscaling capabilities and a better performance when compared to the V1 SKU.

To read more about the V2 SKU, check the [documentation here](https://docs.microsoft.com/en-us/azure/application-gateway/application-gateway-autoscaling-zone-redundant).
