# Configure ExpressRoute Circuit

You can use ExpressRoute circuit to extend your on-premises networks into the Microsoft cloud over a private connection facilitated by a connectivity provider. With ExpressRoute, you can establish connections to Microsoft cloud services, such as Microsoft Azure, Office 365, and Dynamics 365.<br>

## **Recommended documents**

*	Create and modify an ExpressRoute circuit using [portal](https://docs.microsoft.com/azure/expressroute/expressroute-howto-circuit-portal-resource-manager), [PowerShell](https://docs.microsoft.com/azure/expressroute/expressroute-howto-circuit-arm) or [CLI](https://docs.microsoft.com/azure/expressroute/howto-circuit-cli)<br>
*	Routing configuration samples to [setup and manage routing](https://docs.microsoft.com/azure/expressroute/expressroute-config-samples-routing)<br>
*	Create a connection to link a [virtual network to an ExpressRoute circuit](https://docs.microsoft.com/azure/expressroute/expressroute-howto-linkvnet-portal-resource-manager)<br>
*	Learn more about [ExpressRoute workflows](https://docs.microsoft.com/azure/expressroute/expressroute-workflows) for circuit provisioning and circuit states
