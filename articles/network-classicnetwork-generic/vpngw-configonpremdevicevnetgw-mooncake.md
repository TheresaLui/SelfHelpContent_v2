# Configure my on-premises VPN device 

## **Recommended steps**

1. Troubleshoot on-premise connectivity issues using [VPN Diagnostics](data-blade:microsoft_azure_network.networkwatchervpndiagnosticsblade.id.$resourceId)<br>
2. Review the VPN logs (available in storage account) to find any issues with the VPN tunnel or configuration of devices

## **Recommended documents**

* Download [on-premise VPN device configuration script](https://docs.azure.cn/vpn-gateway/vpn-gateway-download-vpndevicescript) for site-to-site VPN connection<br>
* Check Azure [Validated VPN Devices List](https://docs.azure.cn/vpn-gateway/vpn-gateway-about-vpn-devices) to verify whether your 3rd party device is listed<br>
* Check Azure [3rd party device configuration](https://docs.azure.cn/vpn-gateway/vpn-gateway-3rdparty-device-config-overview) for an overview of partner device configurations<br>
* Check [Community suggested solutions](https://docs.azure.cn/vpn-gateway/vpn-gateway-third-party-settings) for third-party VPN or firewall devices used with Azure VPN gateway
