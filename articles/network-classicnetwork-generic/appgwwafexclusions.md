# WAF Exclusions and Global Parameters
Azure Application Gateway WAF provides exclusion lists which allow you to omit certain request attributes from a WAF evaluation. Also, WAF allows you to configure request size limits within lower and upper bounds for example, maximum request body size and file upload limit.
