# Configure my on-premises VPN device 

## **Recommended steps**
1. Troubleshoot on-premise connectivity issues using [VPN Diagnostics](data-blade:microsoft_azure_network.networkwatchervpndiagnosticsblade)<br>
2. Review the VPN logs (available in storage account) to find any issues with the VPN tunnel or configuration of devices

## **Recommended documents**
Check Azure [3rd Party device configuration](https://docs.azure.cn/vpn-gateway/vpn-gateway-3rdparty-device-config-overview) to see if your device is available<br>
Check [Community suggested solutions](https://docs.azure.cn/vpn-gateway/vpn-gateway-third-party-settings) for third-party VPN or firewall devices used with Azure VPN gateway
