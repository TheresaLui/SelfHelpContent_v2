# Configure ExpressRoute Global Reach

You can use ExpressRoute Global Reach to link ExpressRoute circuits together to make a private network between your on-premises networks.<br>

## **Recommended documents**

*	Use [PowerShell](https://docs.microsoft.com/azure/expressroute/expressroute-howto-set-global-reach) to configure ExpressRoute Global Reach<br>
*	ExpressRoute Global Reach [overview](https://docs.microsoft.com/azure/expressroute/expressroute-global-reach) and [FAQ](https://docs.microsoft.com/azure/expressroute/expressroute-faqs#globalreach)<br>
*	[Connect](https://docs.microsoft.com/azure/expressroute/expressroute-howto-linkvnet-portal-resource-manager) virtual network to an ExpressRoute circuit
