# Azure Virtual Network - Azure Virtual Network - Cannot delete VNet because of VM/NIC/IP

If you cannot delete the virtual network, and you see an error message saying a virtual machine (VM), network interface (NIC), or internet protocol (IP) address is in use, then delete that network interface to remove this virtual network. Visit the following link to delete the network interface.

## **Recommended Documents**

- [Delete a network interface](https://docs.microsoft.com/azure/virtual-network/virtual-network-network-interface#delete-a-network-interface)
