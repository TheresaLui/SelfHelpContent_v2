# TLS version with Azure Cache for Redis

The Azure Cache for Redis service supports Transport Layer Security (TLS) when using a supported client. An improperly configured application may result in errors when connecting to the server. Follow the steps in [this article](https://docs.microsoft.com/azure/azure-cache-for-redis/cache-remove-tls-10-11) to configure your application with TLS support.
