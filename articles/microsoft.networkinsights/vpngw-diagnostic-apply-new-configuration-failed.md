# New configuration failed on instance
<!--issueDescription-->
Applying new configuration on virtual network gateway, local network gateway or connection failed.
<!--/issueDescription-->

## **Recommended Steps**

Configuration changes to virtual network gateway, local network gateway or connection may not have been applied successfully. You can check for any errors in recent configuration changes and resubmit with correction. If that doesn't fix the issue, You can try to reset the gateway.
