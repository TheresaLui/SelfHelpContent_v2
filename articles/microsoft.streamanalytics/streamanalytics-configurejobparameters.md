# Configure Job Parameters
Azure Stream Analytics allows you to configure job settings such as scale, output error policy, event ordering, and compatibility level. To learn more about how to configure these settings for your Azure Stream Analytics job, see the recommended documents.

## **Recommended Documents**

* [Configure output error policy](https://docs.microsoft.com/azure/stream-analytics/output-error-policy)
* [Configuring event ordering](https://docs.microsoft.com/azure/stream-analytics/stream-analytics-out-of-order-and-late-events)
* [Configuring scale](https://docs.microsoft.com/azure/stream-analytics/stream-analytics-streaming-unit-consumption#configure-stream-analytics-streaming-units-sus)
