# Develop Functions
Azure Stream Analytics supports user-defined functions written in JavaScript. With the rich set of String, RegExp, Math, Array, and Date methods that JavaScript provides, complex data transformations with Stream Analytics jobs become easier to create. You can invoke these user-defined functions directly from your Stream Analytics query. To learn more about authoring user-defined functions in Azure Stream Analytics job, see the recommended documents.

## **Recommended Documents**

* [Adding a JavaScript user-defined functions](https://docs.microsoft.com/azure/stream-analytics/stream-analytics-javascript-user-defined-functions#add-a-javascript-user-defined-function-in-the-azure-portal)
* [Calling a JavaScript user-defined function in a query](https://docs.microsoft.com/azure/stream-analytics/stream-analytics-javascript-user-defined-functions#call-a-javascript-user-defined-function-in-a-query)
