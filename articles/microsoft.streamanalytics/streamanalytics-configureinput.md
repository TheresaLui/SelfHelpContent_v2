# Configure Input
Stream Analytics has first-class integration with Azure data streams as inputs from Azure Event Hubs, Azure IoT Hub, Azure Blob Storage. Azure Stream Analytics also supports reference data (also known as a lookup table) which is a finite data set that is static or slowly changing in nature, used to perform a lookup or to correlate with your data stream.
