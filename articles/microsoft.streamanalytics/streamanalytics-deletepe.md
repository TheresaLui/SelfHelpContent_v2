# Delete private endpoints in Stream Analytics cluster

You can delete private endpoints in your cluster that you no longer use. Any running jobs that depended on the private endpoints to connect to inputs/output will result in a failed state.

To learn more about private endpoints, see the recommended documents.

## **Recommended Documents**

* [How to create and delete private endpoints](https://docs.microsoft.com/azure/stream-analytics/private-endpoints)
