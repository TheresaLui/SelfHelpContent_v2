# I need help investigating users and accounts

Most users are able to resolve issues investigating user and accounts using the steps below.

## **Recommended Steps**

1. Use the [Detect suspicious user activity with UEBA](https://docs.microsoft.com/cloud-app-security/tutorial-suspicious-activity) tutorial to investigate suspicious users and identify suspicious activities.

1. Once the risky user or account is identified, use the [Investigate risky users](https://docs.microsoft.com/cloud-app-security/tutorial-ueba) tutorial to investigate further.

## **Recommended Documents**

- [Investigate risky OAuth apps](https://docs.microsoft.com/cloud-app-security/investigate-risky-oauth)
- [Investigate anomaly detection alerts](https://docs.microsoft.com/cloud-app-security/investigate-anomaly-alerts)
- [Working with the users and accounts](https://docs.microsoft.com/cloud-app-security/accounts)

If you're still experiencing the issue after reviewing the documentation and configuration, please continue with opening the ticket.
