# Update the VAT ID in CMAT

<!--issueDescription-->

1. In CMAT, navigate to the Tax Information section and click Add.
2. In the Add Tax Information pop-up, select Vat ID from the drop-down.
3. Enter the new Vat ID.
4. Add a comment describing the change you are making.
5. Click the Save button.

Note: If you receive an error in CMAT while or after adding a VAT ID, escalate the issue to PayOps using TSG 2651579.
