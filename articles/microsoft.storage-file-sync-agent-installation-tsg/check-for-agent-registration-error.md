# How to check for agent registration error

Once you are done installing the newest agent on the server, you may not be able to register the server and get an error with the text

~~~powershell

Pre-requisites are missing

Retry server registration after performing the following actions

Install the latest AzureRM PowerShell module by following the documentation: https://go.microsoft.com/fwlink/?linkid=85659

~~~

If the newest agent is installed and you did not get a server registration error and the issue is now resolved, please select the appropriate option below.
