# Getting charged for Storage Accounts

<!--issueDescription-->

Dear customer,<br><br>Please follow the steps given below to investigate the cost incurred:<br><br>* When the Azure Defender is enabled for Storage accounts from Pricing & Setting page it gets enabled for all the Storage Accounts.<br><br>* If you wish to enable Azure Defender for an individual storage account, it can be done by disabling Azure Defender from the  Azure Security Center > Pricing and Setting page and enabling for an individual storage account, for this go to :  The Storage account on which you wish to enable Azure Defender > Security > click on 'Enable Azure Defender for Storage'.<br><br>* We can  check which Storage Accounts were monitored for a particular period of time, for this:<br><br>Go to the Subscription > Cost Analysis > Select the time range > Add the first Filter [ Meter category : advanced threat protection ] > Add the second filter [ Meter subcategory: Storage ] hit enter > in the bottom left hand corner you will see a pie chart, select Resource and it will show you how much we charged per storage account.<br><br><br>* If you do not wish these Storage accounts to be monitored, disable Azure Defender globally by going to the Azure Security Center > Pricing and Setting page.<br><br>* Please refer the following document for pricing details:<br>[**Per Unit Charges**](https://azure.microsoft.com/en-in/pricing/details/azure-defender/)<br><br>* Please refer the following document for more details:<br>   [**Azure Defender for Storage**](https://docs.microsoft.com/en-us/azure/storage/common/azure-defender-storage-configure?tabs=azure-security-center)<br><br> <br><br><br><br><br><br><br><br>*Contact us if you have further queries or concerns related to you charges*<br><br><br><br>Best regards,
