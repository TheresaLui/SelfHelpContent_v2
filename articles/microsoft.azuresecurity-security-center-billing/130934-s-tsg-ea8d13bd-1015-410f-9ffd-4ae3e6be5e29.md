# Getting charged for Servers

<!--issueDescription-->

Dear customer,<br><br>Please follow the steps given below to investigate the cost incurred:<br><br>1. To check the number of Servers which were reporting to a workspace during a particular period of time you can :<br><br><br>* Go to the workspace.<br><br>* Select the period. [ The data retension period need to be considered]<br><br>* Run the following query on the workspace:<br><br><br><br><br><br>Heartbeat<br><br><br><br>| summarize by Computer, ComputerEnvironment, ResourceId<br><br><br><br>| sort by Computer desc<br><br><br><br><br><br><br><br>    **NOTE: We also charge the Agentless machines, the above query can help you to get a list of machines reporting to a particular workspace only, it doesn't show you the machines which does not have the log analytics agent on them or the machine not reporting to that workspace.**<br><br> <br>2. The following query shows you the total number of Units consumed per server in a particular period of time: <br><br><br><br><br>   **NOTE: To get a precise answer please select the Start date, time as 12:00:00 AM and End date, time as 11:59:59 PM and this query needs to be run on individual workspaces to get the details about the servers connected to that workspace.**<br><br><br><br><br><br><br><br>Heartbeat <br><br><br>| summarize by Computer, bin(TimeGenerated, 60m), SubscriptionId<br><br><br>| summarize TotalUnits=count(Computer) by Computer, SubscriptionId<br><br><br>| sort by TotalUnits desc <br><br><br><br><br><br><br><br><br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**Total Cost = Cost per unit (based on the region) x total units**<br><br>  <br><br><br><br><br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;For per unit charges please refer the following link:<br>[**Per Unit Charges**](https://azure.microsoft.com/pricing/details/azure-defender/)<br><br><br><br><br><br>*Contact us if you have further queries or concerns related to you charges*<br><br>Best regards,<br><br><br><br><br><br>
