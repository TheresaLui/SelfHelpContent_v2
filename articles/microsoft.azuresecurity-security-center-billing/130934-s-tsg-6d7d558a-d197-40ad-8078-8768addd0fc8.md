# Getting charged for SQL Database Servers

<!--issueDescription-->

Dear customer,<br><br>Please follow the steps given below to investigate the cost incurred:<br> <br>* When the Azure Defender is enabled for SQL Database Servers from Pricing & Setting page it gets enabled for all the SQL Database Servers.<br>* If you wish to enable Azure Defender for an individual SQL Database Servers, it can be done by disabling Azure Defender from the  Azure Security Center > Pricing and Setting page and enabling for an individual SQL Database Servers, for this go to :  The SQL Database Servers on which you wish to enable Azure Defender > Security > click on 'Enable Azure Defender for SQL'.<br>* If you do not wish these SQL Database Servers to be monitored, disable Azure Defender globally by going to the Azure Security Center > Pricing and Setting page.<br>* Please refer the following document for pricing details:<br>[**Per Unit Charges**](https://azure.microsoft.com/en-in/pricing/details/azure-defender/)<br>* Please refer the following document for more details:<br>   [**Azure Defender for SQL**](https://docs.microsoft.com/en-us/azure/azure-sql/database/azure-defender-for-sql)<br><br> <br>*Contact us if you have further queries or concerns related to you charges*<br><br><br>Best regards,
