# Amount of data ingested for a particular period of time

<!--issueDescription-->

Dear customer,<br><br>To find the amount of data ingested in a particular workspace: <br> <br><br>1. Run the following query on a particular Log Analytics Workspace to see the amount of data ingested per data type.<br><br>**NOTE: Please select the time period before running the query, to get a precise answer please select the Start date, time as 12:00:00 AM and End date, time as 11:59:59 PM**<br><br>**Query:**<br><br>```<br>Usage<br>| where IsBillable == 'TRUE'<br>| project TimeGenerated, DataType, Solution, Quantity, QuantityUnit<br>| summarize DataConsumedPerDataType = sum(Quantity) by QuantityUnit, DataType<br>| sort by DataConsumedPerDataType desc<br>```<br><br><br><br><br>2. Run the following query to find the total data ingested for a particular period of time.<br><br>  **Query:** <br><br>```<br>Usage<br>| where IsBillable == 'TRUE'<br>| project TimeGenerated, DataType, Solution, Quantity, QuantityUnit<br>| summarize DataConsumedPerDataType = sum(Quantity)<br>| sort by DataConsumedPerDataType desc<br>```<br><br><br><br>Depending upon the charges applied per GB or as per plan and per region you can calculate the total cost.<br><br>**Please refer the following links for data ingestion cost:**<br><br>For Monitoring:<br><br>[Azure Monitor pricing](https://azure.microsoft.com/pricing/details/monitor/)<br><br><br><br>*Contact us if you have further queries or concerns related to you charges*<br><br><br>
