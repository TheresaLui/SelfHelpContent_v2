# Spark Job Execution and Performance/Jobs accessing data from Azure Synapse Link for Cosmos DB

## **Recommended Steps**

### **Current Support**

Currently supported Synapse run-times to access Azure Cosmos DB analytical store

* Spark pools - Read, Write (through transactional store), Table, Temporary View

* Serverless SQL pool - Read, View

* Dedicated SQL pool - Not available
