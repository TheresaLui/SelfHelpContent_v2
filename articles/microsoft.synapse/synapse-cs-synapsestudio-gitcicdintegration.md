# Synapse Studio/GIT and CI-CD integration

## **Recommended Steps**

* Please consider the [Best practices for CI/CD](https://docs.microsoft.com/azure/data-factory/continuous-integration-deployment#best-practices-for-cicd)

* Please be aware of the [unsupported features list](https://docs.microsoft.com/azure/data-factory/continuous-integration-deployment#unsupported-features)

* By design, CI/CD publishing does not allow for cherry picking and/or selective publishing subset of changes. To publish individual changes into production environment, please consider Hot fix or QFE

* [Troubleshooting Git integration](https://docs.microsoft.com/azure/data-factory/source-control#troubleshooting-git-integration)
