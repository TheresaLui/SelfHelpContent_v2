# How-To/Portal and Client Tools for SQL on-demand

## **Recommended Steps**

* To obtain connection string for SQL on-demand endpoint, check [Connect to SQL Analytics](https://docs.microsoft.com/azure/synapse-analytics/sql/connect-overview). 

* To connect and query SQL on-demand using Azure Data Studio, follow [steps outlined here](https://docs.microsoft.com/azure/synapse-analytics/sql/get-started-azure-data-studio). Make sure your network does not block port 1433. 

* To connect and query SQL on-demand using SSMS, follow [steps outlined here](https://docs.microsoft.com/azure/synapse-analytics/sql/get-started-ssms). Make sure your network does not block port 1433. 

* To connect Power BI to SQL on-demand, follow [steps outlined here](https://docs.microsoft.com/azure/synapse-analytics/sql/get-started-power-bi-professional). Make sure your network does not block port 1433. 

* If using Synapse Studio, make sure your network does not block port 1443.
