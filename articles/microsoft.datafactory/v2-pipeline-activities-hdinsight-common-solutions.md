# V2 - Pipeline Activities - HDInsight

## **Recommended Steps**

If you received an error message or an error code from HDInsight activities, including Hive, MapReduce, Pig, Spark, and Streaming activity, see the [ADF Troubleshooting Guide](https://docs.microsoft.com/azure/data-factory/data-factory-troubleshoot-guide#hdinsight) to resolve your issues.

If you get an "HDInsight on-demand is not being provisioned" error message, make sure that the service is registered in your subscription.

## **Recommended Documents**

* **If you receive an error code,** see the [ADF Troubleshooting Guide](https://docs.microsoft.com/azure/data-factory/data-factory-troubleshoot-guide#hdinsight) 

* [HDInsight Troubleshooting Guide](https://docs.microsoft.com/azure/hdinsight/hdinsight-troubleshoot-guide) offers a troubleshooting guide for each HDInsight environment
