# Call Azure functions from Azure Logic Apps

## **Recommended Documents**

When you want to run code that performs a specific job in your logic apps, you can create your own function by using Azure Functions. This service helps you create Node.js, C#, and F# functions so you don't have to build a complete app or infrastructure to run code. 

- [How to call Azure functions from Azure Logic Apps](https://docs.microsoft.com/azure/logic-apps/logic-apps-azure-functions)
- [Calling Azure Functions from Azure Logic Apps](https://www.serverlessnotes.com/docs/calling-azure-functions-from-azure-logic-apps)
