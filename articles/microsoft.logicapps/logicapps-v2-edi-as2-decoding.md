# Azure Logic Apps - AS2 Decoding

## **Recommended Steps**

To work with AS2 messages in Azure Logic Apps, you can use the AS2 connector, which provides triggers and actions for managing AS2 communication. AS2 Decode action is used for providing decryption, digital signing, and acknowledgments through Message Disposition Notifications (MDN).

- [Decode AS2 messages](https://docs.microsoft.com/azure/logic-apps/logic-apps-enterprise-integration-as2?WT.mc_id=Portal-Microsoft_Azure_Support#decode)
