# Logic Apps Integration Service Environment usage

## **Recommended Steps**

Integration Service Environment (ISE) is a dedicated private environment where you can run your logic apps and inject that environment inside your Azure Virtual Network (VNET), then you have access to your on-premises and other entities.  

* [Services that can be deployed into a virtual network](https://docs.microsoft.com/azure/virtual-network/virtual-network-for-azure-services#services-that-can-be-deployed-into-a-virtual-network)
