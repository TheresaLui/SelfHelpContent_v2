To create Azure Logic App workflows that integrate across apps, data, cloud services, and systems, use Visual Studio Code and the Azure Logic Apps (Preview) extension to build and locally run [stateful and stateless logic app workflows](https://docs.microsoft.com/azure/logic-apps/create-stateful-stateless-workflows-visual-studio-code#stateful-stateless) in your development environment.
