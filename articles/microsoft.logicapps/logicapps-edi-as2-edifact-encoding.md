# Azure Logic Apps - EDIFACT encoding

## **Recommended Steps**

With the Encode EDIFACT message connector, you can validate EDI and partner-specific properties, generate an XML document for each transaction set, and request a Technical Acknowledgement, Functional Acknowledgment, or both. To use this connector, you must add the connector to an existing trigger in your logic app.

- [Encode EDIFACT messages](https://docs.microsoft.com/azure/logic-apps/logic-apps-enterprise-integration-edifact-encode?WT.mc_id=Portal-Microsoft_Azure_Support#encode-edifact-messages)
- [EDIFACT Encode details](https://docs.microsoft.com/azure/logic-apps/logic-apps-enterprise-integration-edifact-encode?WT.mc_id=Portal-Microsoft_Azure_Support#edifact-encode-details)
