# AS2 Connector - Encoding

## **Recommended Steps**

To work with AS2 messages in Azure Logic Apps, you can use the AS2 connector, which provides triggers and actions for managing AS2 communication. AS2 Decode action is used for providing encryption, digital signing, and acknowledgments through Message Disposition Notifications (MDN), which help support non-repudiation.

- [Encode AS2 messages](https://docs.microsoft.com/azure/logic-apps/logic-apps-enterprise-integration-as2#encode-as2-messages)
