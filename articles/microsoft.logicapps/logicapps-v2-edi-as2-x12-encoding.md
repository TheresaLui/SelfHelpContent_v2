# Azure Logic Apps - X12 encoding

## **Recommended Steps**

With the Encode X12 message connector, you can validate EDI and partner-specific properties, convert XML-encoded messages into EDI transaction sets in the interchange, and request a Technical Acknowledgement, Functional Acknowledgment, or both.

- [Encode X12 messages in Azure Logic Apps with Enterprise Integration Pack](https://docs.microsoft.com/azure/logic-apps/logic-apps-enterprise-integration-x12-encode?WT.mc_id=Portal-Microsoft_Azure_Support)
- [X12 Encode details](https://docs.microsoft.com/azure/logic-apps/logic-apps-enterprise-integration-x12-encode?WT.mc_id=Portal-Microsoft_Azure_Support#x12-encode-details)
