# Managed Connector - SFTP-SSH

To connect to SFTP servers that you can access from the internet by using SSH so that you can work with your files and folders.

## SFTP-SSH Connector Overview

- [Monitor, create, and manage SFTP files by using SSH and Azure Logic Apps](https://docs.microsoft.com/azure/connectors/connectors-sftp-ssh)

## How to Use SFTP-SSH Connector

STFP-SSH is the newest available connector, see overview link above.

- [SFTP](https://docs.microsoft.com/connectors/sftp/) is deprecated.  
- [Considerations](https://docs.microsoft.com/azure/connectors/connectors-sftp-ssh#considerations)
