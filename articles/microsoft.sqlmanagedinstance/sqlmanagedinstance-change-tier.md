# Change tier for a managed instance

## **Recommended Steps**

* Change of service level or performance tier might take some time, depending on the load on SQL Managed Instance and size of data.<br>
* You can view status of operation in your activity log. If the operation takes more than 24 hours, please create a support ticket.
