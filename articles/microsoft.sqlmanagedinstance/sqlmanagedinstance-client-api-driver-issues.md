# Client API driver issues

## **Recommended Steps**

* SQL Managed Instance requires update of client drivers to fully function. The minimum driver versions are:

	* ODBC driver	v17<br>
	* PHP driver	5.2.0<br>
	* JDBC driver	6.4.0.<br>
	* Node.js driver	2.1.1<br>
	* OLE DB driver	18.0.2.0<br>
	* SSMS	17.6<br>
