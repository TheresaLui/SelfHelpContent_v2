# Query Delta Issues

There are multiple Delta query issues covered in this troubleshooter. Please choose the option that best describes the issue.

Some of the following steps would need access to customer's workspace for validations and debugging. Please make sure to get these information from customer:

1. Detailed issue description
2. Subscription ID
3. Workspace ID/URL
4. Cluster Name/ID/URL
5. Was it working fine before? Were there any changes made?
6. Delta table and query details
7. Steps to repro 
8. Customer consent to allow Microsoft/Databricks access workspace and logs for investigation purposes

And follow this [wiki](https://supportability.visualstudio.com/AzureDataBricks/_wiki/wikis/AzureDataBricks.wiki/312341/Access-and-Use-Genie) to access workspace.
