# Issue Category

Customers may face variety of issues with MongoDB error or unexpected result such as:

1. Query Exceeded Max Memory
2. Client connection errors and socket timeout
3. Request timeout errors
4. Request rate exceeded errors
5. MongoDB wire version issues
6. Request is Malformed
7. MongoDB.Driver.MongoWaitQueueFullException
8. Delete with Shard key (MongoDB 3.2)
9. GridFS
10. Storage Size is very large in Mongo
11. Format Exception for mongo query
12. NodeJs Mongoose shardKey
13. Collection Reappearing after delete
