# Check if the Cosmos DB MongoDB account is using 3.6 version

The Exception "Query exceeded the maximum allowed memory usage of 40 MB" is addressed in the Mongo 3.6. The first recommendation is to recommend for Migrating the protocol version to 3.6. After that ensure the client is using the 3.6 endpoint to resolve the issue.
