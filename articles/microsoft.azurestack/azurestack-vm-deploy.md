# Azure Stack  

Azure Stack offers you virtual machines (VMs) as one type of an on-demand, scalable computing resource. You can choose a VM when you need more control over the computing environment than the other choices. 

## **Recommended steps**

1. [Consider the design considerations of your VM](https://docs.microsoft.com/azure/azure-stack/user/azure-stack-compute-overview#what-do-i-need-to-think-about-before-creating-a-vm).<br>
2. Learn the differences between the virtual machine features available in Azure Stack and Microsoft Azure by reading [Considerations for using virtual machines in Azure Stack](https://docs.microsoft.com/azure/azure-stack/user/azure-stack-vm-considerations).<br>
2. Review [Virtual machine sizes supported in Azure Stack](https://docs.microsoft.com/azure/azure-stack/user/azure-stack-vm-sizes).<br>
2. Create your first VM.<br>
    - [Create a-  Windows virtual machine with the Azure Stack portal](https://docs.microsoft.com/azure/azure-stack/user/azure-stack-quick-windows-portal)<br>
    - [Create a Windows virtual machine with the Azure Stack portal](https://docs.microsoft.com/azure/azure-stack/user/azure-stack-quick-linux-portal)<br>
    - [Create a Windows virtual machine by using PowerShell in Azure Stack](https://docs.microsoft.com/azure/azure-stack/user/azure-stack-quick-create-vm-windows-powershell)<br>
    - [Create a Linux virtual machine by using PowerShell in Azure Stack](https://docs.microsoft.com/azure/azure-stack/user/azure-stack-quick-create-vm-linux-powershell)<br>
    - [Create a Windows virtual machine by using CLI in Azure Stack](https://docs.microsoft.com/azure/azure-stack/user/azure-stack-quick-create-vm-windows-cli)<br>
    - [Create a Linux virtual machine by using CLI in Azure Stack](https://docs.microsoft.com/azure/azure-stack/user/azure-stack-quick-create-vm-linux-cli)<br>
