# Azure Stack Virtual Networks

Azure Stack networking has many of the services provided by Azure networking. However, there are some key differences that you should understand before deploying an virtual network:

1. Review [Azure Stack Networking Differences](https://docs.microsoft.com/azure/azure-stack/user/azure-stack-network-differences)
1. Create a virtual network using the [Azure portal](https://docs.microsoft.com/azure/virtual-network/quick-create-portal) or with [Azure CLI](https://docs.microsoft.com/azure/virtual-network/quick-create-cli)
1. [Create, change, or delete a network interface](https://docs.microsoft.com/azure/virtual-network/virtual-network-network-interface).
1. [Add network interfaces to or remove network interfaces from virtual machines](https://docs.microsoft.com/azure/virtual-network/virtual-network-network-interface-vm)
1. [Add, change, or remove IP addresses for an Azure network interface](https://docs.microsoft.com/azure/virtual-network/virtual-network-network-interface-addresses)
