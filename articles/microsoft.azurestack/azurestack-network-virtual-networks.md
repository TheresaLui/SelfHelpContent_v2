# Azure Stack Hub Virtual Networks

Azure Stack Hub networking has a subset of the services provided by Azure networking. There are some differences to consider before deploying a virtual network:

1. Review [Azure Stack Hub Networking Differences](https://docs.microsoft.com/azure/azure-stack/user/azure-stack-network-differences)
1. Create a virtual network using the [Azure portal](https://docs.microsoft.com/azure/virtual-network/quick-create-portal) or with [Azure CLI](https://docs.microsoft.com/azure/virtual-network/quick-create-cli)
