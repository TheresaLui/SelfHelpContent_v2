# Azure Stack  Quotas

Quotas define the limits of resources that a user subscription can provision or consume. For example, a quota might allow a user to create up to five VMs. Each resource can have its own types of quotas.

## **Recommended steps**

1. To view and existing quota, [View an existing quota](https://docs.microsoft.com/en-us/azure/azure-stack/azure-stack-quota-types#view-an-existing-quota).
2. To edit a quota,[Edit a quota](https://docs.microsoft.com/en-us/azure/azure-stack/azure-stack-quota-types#edit-a-quota).
