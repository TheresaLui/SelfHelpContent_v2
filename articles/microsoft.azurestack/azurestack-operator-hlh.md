# Azure Stack Hardware Lifecycle Host

Azure Stack solutions from most hardware vendors include a Hardware Lifecycle Host (HLH), which is a separate physical server that runs hardware management tools from the OEM partner. Azure Stack hardware partners own and maintain guidance for all software and hardware on the hardware lifecycle host.

## **Recommended Steps**

1. Refer to your OEM documentation for details on accessing and using the features of the hardware
2. When replacing Azure Stack hardware components, follow your vendor-specific firmware update process using the HLH to ensure the replaced hardware component has the approved firmware level applied
