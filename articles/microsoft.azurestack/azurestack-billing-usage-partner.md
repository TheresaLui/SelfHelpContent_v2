# Azure Stack CSP billing and usage 

## **Recommended steps**

1. To enable multiple Azure AD tenants to use services in Azure Stack, [enabling multi-tenancy in Azure Stack](https://docs.microsoft.com/azure/azure-stack/azure-stack-enable-multitenancy)
2. For an end user managed by a Cloud Service Provider (CSP), [add tenant for usage and billing to Azure Stack](https://docs.microsoft.com/azure/azure-stack/azure-stack-csp-howto-register-tenants)
3. After you've created a record of your customer in Partner Center, you can sell them subscriptions to products in the catalog using steps to [create, suspend, or cancel customer subscriptions](https://msdn.microsoft.com/partner-center/create-a-new-subscription)
3. If the end customer will manage their own account, [create a guest user in the end customer directory](https://docs.microsoft.com/azure/azure-stack/azure-stack-csp-howto-register-tenants#create-a-guest-user-in-the-end-customer-directory), and send them the details. The end user will then add the guest and elevate the guest permission to **Owner** to the Azure Stack CSP account
