# Azure Stack Storage Configuration

Azure Stack Storage provides blob, table, queue, and account management functionality with Azure-consistent semantics.

## **Recommended Steps**

1. Review key [differences and considerations](https://docs.microsoft.com/azure/azure-stack/user/azure-stack-acs-differences) for Azure Stack Storage Services for the User Environment
2. Follow usage documentation for Azure Stack [Blobs](https://docs.microsoft.com/azure/azure-stack/user/azure-stack-storage-overview#blob-storage), [Tables](https://docs.microsoft.com/azure/azure-stack/user/azure-stack-storage-overview#table-storage), or [Queues](https://docs.microsoft.com/azure/azure-stack/user/azure-stack-storage-overview#queue-storage)
3. Bring user data into Azure Stack using a supported [data transfer tools for Azure Stack storage](https://docs.microsoft.com/azure/azure-stack/user/azure-stack-storage-transfer) 
4. Integrate your applications [with Azure Stack storage development tools](https://docs.microsoft.com/azure/azure-stack/user/azure-stack-storage-dev)
