# I want to move my StorSimple Device Manager from classic portal to the new Azure portal

If you are on the Azure classic portal, perform the following steps to transition to the new Azure portal.

## **Recommended steps**

* Go to [the new Azure portal](http://portal.azure.com).
* Select StorSimple Device Manager.
* Once you select the StorSimple Device Manager, a blade is displayed. Click Migrate at the bottom of this blade.
* When prompted for confirmation, review the impact of this transition and then click Migrate.

[Transition StorSimple 8000 series from classic to new Azure portal](https://docs.microsoft.com/azure/storsimple/)
