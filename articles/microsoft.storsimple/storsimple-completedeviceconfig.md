# I am not able to complete the device configuration in the Azure portal.


## **Recommended steps**
1.  In the local web UI of the virtual array, go to **Troubleshooting** > **Diagnostic tests** and click **Run diagnostic tests**. Validate that the device is configured properly and resolve any reported issues.
2.  Ensure that the correct ports have been opened on the device.

[StorSimple port configuration and networking requirements](https://aka.ms/storsimple-troubleshoot-nwreq)<br>
