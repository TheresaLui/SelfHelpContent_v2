# Azure Information Protection client - Installing, Onboarding, or Decommissioning - OnPrem Onboarding

## **Recommended Steps**

1. If your issue is related to installing the Azure Information Protection client or scanner, select the appropriate support topic to review relevant Self Help content.

2. If your issue is related to using a hybrid configuration, review [Does Azure Information Protection support on-premises and hybrid scenarios?](https://docs.microsoft.com/azure/information-protection/faqs#does-azure-information-protection-support-on-premises-and-hybrid-scenarios)
