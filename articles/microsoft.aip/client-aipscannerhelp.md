# Azure Information Client - AIP Scanner 

## How to export Azure Information Protection scanner logs

1. Locate the following folder for the scanner service account: %localappdata%\Microsoft\MSIP\Scanner.
2. Send the files in this folder to a compressed (zip) folder.
3. Attach the resulting .zip file to this service request.
