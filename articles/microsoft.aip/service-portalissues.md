# Azure Information Protection service - Azure portal issues

## Recommended troubleshooting steps

1. If you experience issues accessing the Azure Information Protection portal, verify the following:
   - You are using one of the following [administrator roles](https://docs.microsoft.com/azure/active-directory/active-directory-assign-admin-roles-azure-portal): Information Protection Administrator, Security Administrator, or Global Administrator.
   - You are not using a Microsoft account or a guest account, but an Azure Active Directory administrator account.
   - Your account is not configured for Azure Active Directory Conditional Access.

If you still experience the issue, take a screenshot of the problem and attach the screenshot to this ticket.
