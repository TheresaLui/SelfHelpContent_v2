# Create or update MariaDB server failed because the input storage size does not meet requirements.

<!--issueDescription-->
Thank you for contacting Microsoft support team. We have detected that your storage size does not meet the requirement of your MariaDB pricing tier.
<!--/issueDescription-->

## **Recommended Steps**

Please find the storage limits of your pricing tier from [Pricing tiers in Azure Database for MariaDB - Single Server](https://docs.microsoft.com/azure/mariadb/concepts-pricing-tiers).
