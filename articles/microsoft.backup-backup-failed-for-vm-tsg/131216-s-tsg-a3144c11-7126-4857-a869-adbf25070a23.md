# UserErrorKeyVaultPermissionsNotConfigured final

<!--issueDescription-->

We identified that your backup operation failed due to the backed up VM that does not have required permission to access the key vault or the Key used for encrypting the VM is not a standalone key but part of the certificate that Azure Backup supports.<br><br>
