# Welcome to the TSG for ZRS migration scenarios

## TSG Scope

### This TSG covers scenarios for the following support topics:
- Routing Azure Storage Management\Data Migration\Help me choose a data migration solution
- Routing Azure Storage Blob\Data Migration\Help me choose a data migration solution
- Routing Azure Storage File\Data migration\Help me choose a data migration solution

###  When to use this TSG
- Use this TSG for a request to migrate a storage account to ZRS.
- As you follow through the steps in this TSG, maintain notes of each command run and output generated.
- Should you run into any difficulties, this information will be critical to determine next steps.
