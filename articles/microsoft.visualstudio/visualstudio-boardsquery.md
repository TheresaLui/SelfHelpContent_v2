# Work Item Query Issues

## **Recommended Steps**

Are you facing one of these common problems?

* I am unable save a query to a folder in Shared Queries

  - Check to make sure you have the right permissions on the folder. If you are attempting to edit an existing query, check permissions on that specific query. [Learn more](https://docs.microsoft.com/azure/devops/boards/queries/set-query-permissions?view=azure-devops)

* We need to know how to export query results to Excel

  - Use the Office Client and Excel to export and bulk update work items. [Learn more](https://docs.microsoft.com/azure/devops/boards/backlogs/office/bulk-add-modify-work-items-excel?view=azure-devops&tabs=agile-process)

  - Export query results to a csv file. [Learn more](https://docs.microsoft.com/azure/devops/boards/queries/import-work-items-from-csv?view=azure-devops#export-list-as-csv)

* How do I bulk import or update work items?

  - Use the csv import to import new items or updated items. [Learn more.](https://docs.microsoft.com/azure/devops/boards/queries/import-work-items-from-csv?view=azure-devops)

  - Modify many work items using the bulk edit feature. [Learn more](https://docs.microsoft.com/azure/devops/boards/backlogs/bulk-modify-work-items?view=azure-devops)

* How to create a query that get work items across several projects?

  - Use the quick search to find work items across projects. [Learn more](https://docs.microsoft.com/azure/devops/project/search/work-item-search?view=azure-devops)

  - [Query across projects](https://docs.microsoft.com/azure/devops/boards/queries/using-queries?view=azure-devops#query-across-projects)

* I have queries that are running slow, how do I make them faster?

  - Create high-performing queries. [Learn more](https://docs.microsoft.com/azure/devops/boards/queries/high-performing-queries?view=azure-devops)
