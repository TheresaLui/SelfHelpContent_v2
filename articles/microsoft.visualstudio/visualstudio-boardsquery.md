# Work Item Query Issues

Resolve common issues with work item queries using the following recommendations.

## **Recommended Steps**


* **I am unable save a query to a folder in Shared Queries**

  - Make sure that you have the right permissions on the folder. If you are attempting to edit an existing query, check permissions on [that specific query](https://docs.microsoft.com/azure/devops/boards/queries/set-query-permissions?view=azure-devops).

* **How can I export query results to Excel**

  - Use the Office Client and Excel to [export and bulk update work items](https://docs.microsoft.com/azure/devops/boards/backlogs/office/bulk-add-modify-work-items-excel?view=azure-devops&tabs=agile-process)

  - Export query results to a [csv file](https://docs.microsoft.com/azure/devops/boards/queries/import-work-items-from-csv?view=azure-devops#export-list-as-csv)

* **How do I bulk import or update work items?**

  - Use the csv import to [import new items or updated items](https://docs.microsoft.com/azure/devops/boards/queries/import-work-items-from-csv?view=azure-devops)

  - Modify many work items using the [bulk edit feature](https://docs.microsoft.com/azure/devops/boards/backlogs/bulk-modify-work-items?view=azure-devops)

* **How can I create a query to get work items across several projects?**

  - Use the quick search to [find work items across projects](https://docs.microsoft.com/azure/devops/project/search/work-item-search?view=azure-devops)

  - [Query across projects](https://docs.microsoft.com/azure/devops/boards/queries/using-queries?view=azure-devops#query-across-projects)

* I **have queries that are running slow, how do I make them faster?**

  - Create [high-performing queries](https://docs.microsoft.com/azure/devops/boards/queries/high-performing-queries?view=azure-devops)
