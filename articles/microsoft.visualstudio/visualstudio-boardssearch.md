# Work Item Search

Resolve issues with lost work items using the following steps. 

## **Recommended Steps**

* **I can't find a work item that I know exists. How can I find it?**

  - Check the [recycle bin](https://docs.microsoft.com/azure/devops/boards/backlogs/remove-delete-work-items?view=azure-devops)to see of the work item was deleted

  - Try using [work item search](https://docs.microsoft.com/azure/devops/project/search/work-item-search?view=azure-devops)
