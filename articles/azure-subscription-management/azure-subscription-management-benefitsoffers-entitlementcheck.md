# entitlement check

## **Recommended Steps**

### Activate Azure Sponsorship

This offer for a limited sponsored Microsoft Azure account is **available by email invitation only for limited customers selected by Microsoft**. If you are entitled to the Microsoft Azure Sponsorship offer, you would have received an email invitation to accept this offer for your Account Id.<br>

  1. Go to [Microsoft Azure Sponsorship Offer](https://azure.microsoft.com/offers/ms-azr-0036p/) and click **Activate** <br>
  2. To cancel your current subscription, please refer [how to cancel Azure subscription](https://docs.microsoft.com/azure/billing/billing-how-to-cancel-azure-subscription)<br>
  3. To check your sponsorship Balance, refer [Check your Balance](https://www.microsoftazuresponsorships.com/Balance) <br>

Learn more:

* Azure Sponsorship: [Microsoft Azure Sponsorship Offer](https://azure.microsoft.com/offers/ms-azr-0036p/)
* [Azure entitlement levels](https://azure.microsoft.com/support/plans/)
* [CSP Program](https://docs.microsoft.com/partner-center/csp-overview)

### Azure Benefit for Microsoft Partner Network (MPN)
Click [Activate](https://azure.microsoft.com/offers/ms-azr-0025p/) to activate your MPN offer

### Azure Hybrid Benefit
The Azure Hybrid Benefit is a pricing benefit for customers who have licenses with Software Assurance, which helps maximize the value of existing on-premises Windows Server and/or SQL Server license investments when migrating to Azure.

### Products are eligible for the Azure Hybrid Benefit
Azure Hybrid Benefit is available to customers who own eligible SQL Server and/or Windows Server licenses with active Software Assurance or the equivalent qualifying subscription licenses.

   * Windows Server Datacenter edition with Software Assurance
   * Windows Server Standard Edition with Software Assurance
   * SQL Server Enterprise Core with Software Assurance
   * SQL Server Standard Core with Software Assurance

The offer is available in all [Azure cloud regions](https://azure.microsoft.com/global-infrastructure/regions/)

Customers eligible for Azure Hybrid Benefit can use this benefit in Azure Dedicated Host (more details in Windows Server and SQL Server sections).<br>
Learn more: [Azure Hybrid Benefit](https://azure.microsoft.com/pricing/hybrid-benefit/)
