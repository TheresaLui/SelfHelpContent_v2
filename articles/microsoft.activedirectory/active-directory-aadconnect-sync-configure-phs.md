# Configure password hash synchronization

## **Recommended Steps**

Here are steps to configure password hash synchronization using Azure AD Connect:

* If you **are installing** Azure AD Connect, follow the steps to [configure password hash synchronization](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-password-hash-synchronization#enable-password-hash-synchronization)

* If you have **already installed** Azure AD Connect, follow the steps to [modify password hash synchronization](https://docs.microsoft.com/azure/active-directory/hybrid/plan-connect-user-signin#changing-the-user-sign-in-method)
