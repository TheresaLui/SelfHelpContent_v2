# Azure AD Sign-in logs

## **Recommended steps**

### **Troubleshooting download issues**

* If you don't see all the sign-ins for the date range you selected, you can download up to 5000 rows (sorted by most recent) of sign-ins from the Azure portal.<br>
* If you want to download more, open the Sign-ins blade and select the **Script** option to download a PowerShell script that can give you the results in .csv format
