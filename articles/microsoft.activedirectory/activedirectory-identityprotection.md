# Identity Protection

## **Recommended steps**

Try the following:
* Ensure that the user signs in with global administrator, security administrator, or security reader permissions.
* Ensure that you have Azure AD Premium P2 or Enterprise Mobility + Security E5.
* Ensure that you have [onboarded to Identity Protection](https://aka.ms/IPSignUp).
