# With QnA Maker

## **Recommended Steps**

* Cognitive Services provides two natural language processing services, [LUIS](https://docs.microsoft.com/azure/cognitive-services/luis/)and [QnA Maker](https://docs.microsoft.com/azure/cognitive-services/qnamaker/), each with a different purpose

* Dispatch is a CLI tool that allows you to create an application that combines LUIS and QnA Maker together

* Understand when to [use each service and how they compliment each other](https://docs.microsoft.com/azure/cognitive-services/luis/choose-natural-language-processing-service#when-to-use-each-service)

* Implement and learn more about the [best practices](https://docs.microsoft.com/azure/cognitive-services/luis/choose-natural-language-processing-service#best-practices) for each service

* Learn more about using [Dispatch tool](https://docs.microsoft.com/azure/bot-service/bot-builder-tutorial-dispatch?view=azure-bot-service-4.0&tabs=cs) and [how to install](https://github.com/microsoft/botbuilder-tools/tree/master/packages/Dispatch) it
