# Storage Tiers for Azure Files

## **Recommended Documents**    

A file share in a general purpose v2 (GPv2) storage account can contain transaction optimized, hot, or cool file shares (or a mixture thereof). 
Transaction optimized shares are available in all Azure regions, but hot and cool file shares are only available in a subset of regions. 
You can create a hot or a cool file share using the Azure PowerShell preview module or the Azure CLI

- [Overview on Storage tiers for Azure Files](https://docs.microsoft.com/azure/storage/files/storage-files-planning#storage-tiers)<br>
- [Enabling Storage tiers using Powershell or CLI](https://docs.microsoft.com/azure/storage/files/storage-how-to-create-file-share?tabs=azure-portal#create-a-hot-or-cool-file-share)<br>
- [Storage tiers availability by region](https://docs.microsoft.com/azure/storage/files/storage-files-planning#storage-tiers)<br>
