# Enabling AD DS Authentication for Azure Files

:::Section Recommended Solution:::

The below set of articles and videos focus on configuring AD DS Authentication with Azure Files
1) Before you enable AD DS authentication for Azure Files, please refer documentation in the Prerequistes section given below 
2) For specific networking configurations, please refer documentation in the Networking configuration given below
3) After completing the pre-requisites and networking configuration, refer to the documentation in the Step by Step instructions for detailed steps on how to enable AD DS authentication for Azure Files
4) Video on enabling AD DS authentication for Azure Files with Private endpoints and DNS Forwarding
5) Video and documentation on enabling AD DS authentication for Azure Files with Windows Virtual Desktop
