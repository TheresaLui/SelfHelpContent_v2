# Issues with diagnostic logs

## **Recommended Steps**

Most of the customers resolved the questions about diagnostic logs on their own with the documents below.

- With [Azure diagnostic logs](https://docs.microsoft.com/azure/cdn/cdn-azure-diagnostic-logs) you can view core analytics and save them into one or more destinations.
	* [Enable logging with the Azure portal](https://docs.microsoft.com/azure/cdn/cdn-azure-diagnostic-logs#enable-logging-with-the-azure-portal)<br>
	* [Enable logging with PowerShell](https://docs.microsoft.com/azure/cdn/cdn-azure-diagnostic-logs#enable-logging-with-powershell)<br>
	* [Consuming diagnostics logs from Azure Storage](https://docs.microsoft.com/azure/cdn/cdn-azure-diagnostic-logs#consuming-diagnostics-logs-from-azure-storage)<br>
	* [Consuming diagnostics logs from a Log Analytics workspace](https://docs.microsoft.com/azure/cdn/cdn-azure-diagnostic-logs#consuming-diagnostics-logs-from-a-log-analytics-workspace)<br>
