# Transfer data with AzCopy v10

<!--issueDescription-->
AzCopy v10 is the command-line utility for copying data to/from Microsoft Azure Blob and File storage, which offers a redesigned command-line interface and new architecture for high-performance reliable data transfers. This version of AzCopy has several performance improvements over prior versions. When facing performance issues with copy operations, please use AzCopy v10 to see if the issues are resolved. <!--/issueDescription-->

Please refer to [Transfer data with AzCopy v10](https://docs.microsoft.com/azure/storage/common/storage-use-azcopy-v10) for more information. If AzCopy v10 does not resolve the issues, please contact customer support.
