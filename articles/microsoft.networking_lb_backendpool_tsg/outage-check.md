# How to check for outages in the region

## **Recommended Steps**

* Check for any outages in the region. If possible, please have the customer bring the VM online and ensure programming is complete
* If there are no outages, then engage Compute to help troubleshoot the issue further
* If the backend pool contains Network Virtual Appliances (NVAs) reference this [NVA TSG](https://aka.ms/anpnvawiki) for additional troubleshooting steps.

## **Recommended Documents**

* [Azure Regional Status](http://azure.microsoft.com/status/)
* [Azure SIE Playbook](https://aka.ms/azsiepb)
