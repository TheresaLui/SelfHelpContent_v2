# Errors and Exceptions/403 Forbidden or Quota Exceeded

## **Recommended steps**
* This may happen if you have exceeded the API call quota. If you are on Free or Basic tier, upgrading to Standard tier will increase your quotas. <br>
* Check to see if you are calling an API that is only available for higher tiers (e.g. import/export feature is available for Standard tier; Per Message Telemetry feature is available for Standard tier; etc)
