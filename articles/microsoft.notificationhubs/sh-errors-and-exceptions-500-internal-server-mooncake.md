# Errors and Exceptions/500 Internal Server

## **Recommended steps**
Notification Hub is a multi-tenant environment. If there are very few errors, retry should help.<br>
If there are lots of internal server errors, check the [dashboard](https://azure.microsoft.com/zh-cn/status/) to see if there are any outages. <br>
