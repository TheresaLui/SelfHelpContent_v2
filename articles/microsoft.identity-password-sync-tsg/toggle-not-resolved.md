# Collect AADConnect wizard logs

* Open Azure AD Connect Wizard from the Desktop icon, click Configure, Troubleshoot, then Launch.
* On the PowerShell menu, click ‘3’ to Collect General Diagnostics Report.
* If you only want to get Azure AD Connect Wizard trace logs, open the folder %ProgramData%\AADConnect in Windows/File Explorer and create a zip file with all the contents.
