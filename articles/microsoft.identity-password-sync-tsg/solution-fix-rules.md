# Fix sync rules

* To fix a Sync rule open the Synchronization Rules Editor, select the sync rule and click ‘Edit’.
* To fix a export or sync error, first identity what is the cause for the error and then fix the issue on the respective on-premises object.
* To fix OU Filtering open Azure AD Connect Wizard from the Desktop and click Configure, then Customize synchronization options. 
* Provide Azure AD and local AD credentials and proceed to Domain/OU Filtering page.
