# Difficulties using Azure Data Lake SDK

You can use the Azure Data Lake SDK (.Net) to operate ADL jobs, accounts, data sources, and users. There are a number of prerequisites, as seen below.

## **Recommended Steps**

1. Check if there are outages in the region where this job is running on [Azure status page](https://status.azure.com/status)<br>
2. Make sure that one of Visual Studio 2019, Visual Studio 2018, Visual Studio 2015, Visual Studio 2013 update 4, or Visual Studio 2012 with Visual C++ Installed are [installed](https://docs.microsoft.com/visualstudio/install/install-visual-studio?view=vs-2019)<br>
3. Ensure Microsoft Azure SDK for .NET version 2.5 or above is [installed](https://www.microsoft.com/web/downloads/platform.aspx)<br>
4. [Install required NuGet Packages](https://docs.microsoft.com/azure/data-lake-analytics/data-lake-analytics-manage-use-dotnet-sdk#install-nuget-packages)<br>
