# Microsoft Graph Excel API issues

## **Recommended Steps**

Check the answers already available in Stack Overflow for [Excel APIs](https://stackoverflow.com/search?q=%5Bmicrosoft-graph%5D+excel+isanswered%3Ayes+views%3A50) or craft your own, more specific search query. If you can't find a solution to your problem, [ask a new question on Stack Overflow](https://stackoverflow.com/questions/ask) and tag with *microsoft-graph*.

**Authentication or authorization issues** 

- If your app is **unable to acquire tokens** to call Microsoft Graph Excel APIs, pick the **Problem with getting an access token (Authentication)** Graph API category to get more specific help and support on this topic
- If your app is **receiving 401 or 403 authorization errors** when calling Microsoft Graph Excel APIs, pick the **Getting an access denied error (Authorization)** Graph API category to get more specific help and support on this topic

**Querying or provisioning resource issues**

- If your app is **unable to provision or query resources** using Microsoft Graph Excel APIs, pick the **Problem with querying or provisioning resources** Graph API category to get more specific help and support on this topic
