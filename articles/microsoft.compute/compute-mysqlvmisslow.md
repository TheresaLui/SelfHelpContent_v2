# My SQL Server VM is slow

## **Recommended Steps**

Try following steps to diagnose and mitigate SQL VM performance issues.

1. [Use Premium Storage documentation](https://azure.microsoft.com/documentation/articles/storage-premium-storage/) and enable read caching (set Host caching option to ReadOnly)<br>
2. [Review performance best practices for SQL Server in Azure Virtual Machines](https://azure.microsoft.com/documentation/articles/virtual-machines-windows-sql-performance/)<br>
3. [Troubleshoot generic SQL Server workload performance issues](https://docs.com/ajith-krishnan/8919/sql-server-performance-troubleshooting-approach)
