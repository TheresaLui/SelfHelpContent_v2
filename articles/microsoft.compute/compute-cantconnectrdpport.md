# I can't connect to my Windows VM

## **Recommended documents**

1. Click [here](https://review.docs.microsoft.com/azure/virtual-machines/troubleshooting/troubleshoot-rdp-connection?branch=pr-en-us-54175) to troubleshoot common RDP connection issues.

4. [Serial Console guide](https://docs.microsoft.com/azure/virtual-machines/troubleshooting/serial-console-windows)
