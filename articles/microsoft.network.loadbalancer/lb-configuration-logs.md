# Azure Load Balancer Logs

## **Recommended Steps**
* Standard Load Balancer currently does not support logs. Learn more about how to set up [metrics and alerts](https://docs.microsoft.com/azure/load-balancer/load-balancer-standard-diagnostics) on Azure Standard Load Balancer.
* To learn more about types of logs you can enable, visit [Azure Monitor Logs for Public Basic Load Balancer](https://docs.microsoft.com/azure/load-balancer/load-balancer-monitor-log).
* Learn how to 
1. [Archive to storage account logs](https://docs.microsoft.com/azure/load-balancer/load-balancer-monitor-log#archive-to-storage-account-logs)
2. [Stream to an event hub](https://docs.microsoft.com/azure/load-balancer/load-balancer-monitor-log#stream-to-an-event-hub-1)
3. [Send to Log Analytics](https://docs.microsoft.com/azure/load-balancer/load-balancer-monitor-log#send-to-log-analytics-1)
