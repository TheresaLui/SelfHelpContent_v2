# Check if the customer is facing issues with Hybrid Module installation or with running the join command. 

* Module Install issues = Issues while running **".\CopytoPSPath.ps1"** command or issues while running **"Import-Module -name AzFilesHybrid"** command. 

* Domain Join issues = Issues while running **"Join-AzStorageAccountforAuth"** command.
