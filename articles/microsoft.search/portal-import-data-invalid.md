# Issue with index data when using the import data wizard

## **Recommended Steps**

If you are having trouble with the data in the index generated by the import data wizard, you may want to ensure that:

1. Your data source definition is valid, the wizard is able to read a file/record from the data source 
1. The files/records in your data source are consistent with the index data
1. Set your [parsing mode](https://docs.microsoft.com/azure/search/search-howto-index-json-blobs#parsing-modes) to use appropriate document cracker
