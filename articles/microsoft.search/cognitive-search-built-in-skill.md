# Issue with a built-in cognitive skill

## **Recommended Steps**

If you are having trouble with a built-in cognitive skill:

1. Confirm that the names and sources of the inputs to your skill are valid. It can be useful to draw out the structure of your annotations to confirm that the input to a skill is correct. More information about how to reference annotations in a skillset can be found [here](https://docs.microsoft.com/azure/search/cognitive-search-concept-annotations-syntax).
1. Confirm that the name and target name for the output is valid
1. Check that the right context has been set. More information about the context can be found [here](https://docs.microsoft.com/azure/search/cognitive-search-defining-skillset#add-built-in-skills).
