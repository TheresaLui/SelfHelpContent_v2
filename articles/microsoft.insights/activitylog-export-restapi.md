# <-- configuring-export-restapi -->
You can use REST API to configure subscription diagnostic settings to continuously export your Activity Logs to Log Analytics workspace, Azure Storage account, or Azure Event Hub.

## **Recommended Steps**

* Use the guide of how to export [Diagnostic setting](https://docs.microsoft.com/rest/api/monitor/diagnosticsettings) using [Azure Monitor REST API](https://docs.microsoft.com/rest/api/monitor/)
