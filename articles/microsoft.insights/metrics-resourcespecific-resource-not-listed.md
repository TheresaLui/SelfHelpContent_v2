# <-- metrics-for-resource-type-other -->

A detailed list of metrics for each Azure Service can be found [here](https://docs.microsoft.com/azure/azure-monitor/platform/metrics-supported). The page contains a short description, lists any dimensions the metric contains, and provide metadata (like units, and aggregation types) for every metric.

## **Recommended Steps**

1. Navigate to the [List of Supported Metrics]((https://docs.microsoft.com/azure/azure-monitor/platform/metrics-supported)).
1. Find, the resource type that you are interested in (use the right navigation menu, or **Ctrl+F** in browser to search)
1. Review the descriptions, dimensions, and metadata for the metrics that resource type exposes.
