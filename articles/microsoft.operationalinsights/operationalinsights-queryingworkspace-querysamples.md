# Query samples

Many query samples are available through Log Analytics itself - <br/>
1. Basic examples are available when you open a new tab, or open the *Help* menu, and select *Examples*
2. Additional query examples and solution queries are available through the Query Explorer.

Additional examples are available in the documentation articles listed below. You can use the Log Analytics [demo environment](https://portal.loganalytics.io/demo#) to try out any of these examples on real data.
