# Help with Budgets

Cloudyn helps forecast future spending using historical usage and spending data. Cloudyn reports can be used to view all cost projection data

**Prerequisites**

  * You must have an Azure account.
  * You must have either a trial registration or paid subscription for Cloudyn.


Learn more on how to manage budgets : [Manage Budgets](https://docs.microsoft.com/azure/cost-management/manage-budgets)<br>

Learn more on how to forecast future spending  : [Forecast spending](https://docs.microsoft.com/azure/cost-management/tutorial-forecast-spending)
