# Setup help for Azure EA

**Automate renewing Azure EA API Access key**<br>

Unfortunately there is no automation process for API key renewal. There only option to generate an API key in EA portal and update it in Cloudyn portal manually. Cloudyn has been migrated with Azure Cost Management for all EA and web direct accounts. Oct 2019 onwards all EA and Web direct Cloudyn account will be retired. It is recommended to use Azure Cost Management instead of Cloudyn.<br>

Learn more: [FAQ: How to restore the connection between Cloudyn and Azure when your Azure EA API Key expires](https://social.msdn.microsoft.com/Forums/en-US/58c2caa1-6eaa-4e1f-ac4f-1dee9a62c07f/faq-how-to-restore-the-connection-between-cloudyn-and-azure-when-your-azure-ea-api-key-expires?forum=Cloudyn)<br>
Learn more: [FAQ: How to activate 'un-activated' accounts with ARM credentials to view Azure usage data in Cloudyn](https://social.msdn.microsoft.com/Forums/en-US/3eeba735-b118-4b7c-82a9-837a1fce5403/faq-how-to160activate-quotunactivatedquot-accounts-with-arm-credentials-to-view-azure-usage?forum=Cloudyn)<br>
