# How to check the CPU \ Memory Utilization of the Gateway using Visual Debugging

## **Recommended Steps**

* Check the CPU \ Memory Utilization of the Gateway via ASC --> Select the Gateway --> under Visual Debugging --> Open TenantSystemStats
* System counters for tenant like NetworkInterfaceStats, ProcessorTime, MemoryUsage and more.
