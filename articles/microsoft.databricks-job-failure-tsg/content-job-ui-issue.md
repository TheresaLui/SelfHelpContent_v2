# Job UI issue

If Job UI is not loading or customer is facing error while listing the jobs in the UI,this usually means having outages with Databricks services. Please refer to [Azure Databricks Status](https://status.azuredatabricks.net) page.

1. If issue is already identified, please follow up with team for update, and provide customer with next ready message along with identified issue update.
2. If issue is not identified, this needs to involve Databricks team to check the impacted backend service. Please follow this [Wiki Link](https://supportability.visualstudio.com/AzureDataBricks/_wiki/wikis/AzureDataBricks.wiki/312121/Engaging-Databricks-Support) to create the Salesforce ticket, and provide them with the next ready message.

**Note: make sure to specify the symptom customer is facing in the ready message since this issue could result in different symptoms. (Be Specific)**
