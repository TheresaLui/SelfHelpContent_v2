# QnA Components

### **Frequently asked questions**

* **Can we share the QnA Maker Azure Search with other services?**<br>
Yes, you can share. Please find the recommendations to avoid impact to QnA Maker service [here](https://docs.microsoft.com/azure/cognitive-services/qnamaker/concepts/azure-resources#share-services-with-qna-maker).

* **How to configure Azure Search to handle more RPS (Request Per Second)?**<br>
You have full control to scale up/down the azure search as per your RPS needs. You can find the recommended settings [here](https://docs.microsoft.com/azure/cognitive-services/qnamaker/concepts/azure-resources#recommended-settings).
