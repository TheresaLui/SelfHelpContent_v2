# How to create a genie access ticket

1. Go to Salesforce portal to submit ticket: [Submit Request](https://help.databricks.com/s/submitrequest)
2. Fill all required details making sure to have the following:
    1. Business Impact - Low
    2. Component - Genie access request
    3. Check the permission to access Databricks instance.
    4. Workspace ID
    5. Cluster ID
    6. Requestor Organization
    7. Requestor Subscription ID
    8. Requestor Ticket ID
3. Submit request and copy ticket number to use it afterwards.
