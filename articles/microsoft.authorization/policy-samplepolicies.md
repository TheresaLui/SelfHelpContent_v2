# Azure Policy - Policy sample policies

## **Recommended Steps**

* **Resource tags policies**
You can enforce a tag and its value using this [sample](https://docs.microsoft.com/azure/governance/policy/samples/enforce-tag-value).

You can follow [this tutorial](https://docs.microsoft.com/azure/governance/policy/tutorials/govern-tags) explaining how to manage tag governance.  

Please refer [here](https://github.com/Azure/azure-policy/tree/master/samples/Tags) for examples of using the new Modify effect on tags to add a tag or addOrReplace a tag. 

You can get a list of policy samples in [our documentation](https://docs.microsoft.com/azure/governance/policy/samples/allowed-locations), [Azure-policy GitHub Repo](https://github.com/Azure/azure-policy) and [Community Policy repo](https://github.com/Azure/Community-Policy). 

* **VM Extensions policies**

You can use the [sample](https://docs.microsoft.com/azure/virtual-machines/extensions/extensions-rmpolicy-howto-cli) to restrict extensions installed to linux vms. 

You can get a list of policy samples in [our documentation](https://docs.microsoft.com/azure/governance/policy/samples/allowed-locations), [Azure-policy GitHub Repo](https://github.com/Azure/azure-policy) and [Community Policy repo](https://github.com/Azure/Community-Policy). 

* **Enforce NSG on a sub net**

You can get samples for Network [here](https://github.com/Azure/azure-policy/tree/master/samples/Network). 

You can get a list of policy samples in [our documentation](https://docs.microsoft.com/azure/governance/policy/samples/allowed-locations), [Azure-policy GitHub Repo](https://github.com/Azure/azure-policy) and [Community Policy repo](https://github.com/Azure/Community-Policy).
