# Kubernetes applications IOT

Use the IoT Edge module to deploy a stateless application on your Azure Stack Edge device.

## **Recommended Steps**

1.	Configure access: Use the PowerShell runspace to create a user, create a namespace, and grant user access to that namespace.
2.	Configure storage: Use the Azure Stack Edge resource in the Azure portal to create persistent volumes by using either static or dynamic provisioning for the stateful applications that you deploy.
3.	Configure networking: Use the services to expose applications externally and within the Kubernetes cluster.
