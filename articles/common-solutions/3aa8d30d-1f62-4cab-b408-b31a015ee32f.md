# Unable to SSO into Confluent Cloud SaaS portal

## **Recommended Steps**

* Please ensure the right Azure AD email is used and that consent is provided to access the Confluent portal when you navigate to the Confluent SSO URL
* Please contact [Confluent Support](https://support.confluent.io/) if issue persists
