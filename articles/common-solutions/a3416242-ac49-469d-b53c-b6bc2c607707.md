# Azure Sentinel - Entity Behavior - Enabling Entity Pages

## **Recommended Steps**

1.	Make sure you followed the prerequisites for enabling entity pages described [here](https://docs.microsoft.com/azure/sentinel/enable-entity-behavior-analytics#prerequisites)

2.	Enable the entities sync from Azure Active Directory in order to create user profiles

3.	After enabling the entity pages, you will be forwarded to the ‘Search page’ experience
	1.	Try searching for a user / host 
	2.	Click on the top users / host to pivot to the entity page
