# Security Defaults

Most users are able to resolve their Security Defaults issues using the steps below.

## **Recommended Steps**

If your tenant was created after October 21, 2019, it might have Security Defaults enabled. [Here's how it works](https://docs.microsoft.com/azure/active-directory/fundamentals/concept-fundamentals-security-defaults) 

Do you want to customize Security Defaults protection? [Learn how using Conditional Access](https://docs.microsoft.com/azure/active-directory/conditional-access/concept-conditional-access-policy-common#typical-policies-deployed-by-organizations)
