# attaching new disk

You can manage your VM and perform tasks like attaching a new or existing disk using a PowerShell cmdlet or the Azure portal. 

## **Recommended Steps**

1. Connect to the Azure Resource Manager on your device.

2. To create a VM image from the managed disk, run the following command:<br>
   ```Replace <Disk name>, <OS type>, and <Disk size> with real values```
