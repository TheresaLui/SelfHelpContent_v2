# Peering setup failed with error Bad Argument

Request failed with bad argument.

## **Recommended Steps**

1. This usually happens when an incorrect data is provided during peering create or update request. Microsoft might already have a peering with ASN at given location.
2. Please refer to [Convert Legacy Exchange Peering](https://docs.microsoft.com/azure/internet-peering/howto-legacy-exchange-portal) or [Convert Legacy Direct Peering](https://docs.microsoft.com/azure/internet-peering/howto-legacy-direct-portal) for more details about legacy peering conversion
