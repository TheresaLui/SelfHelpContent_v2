# Authentication issues

Resolve Azure authentication issues using the following steps.

## **Recommended Steps**

1. If you are setting up a new environment, review the requirements section of the [Overview](https://docs.microsoft.com/azure/virtual-desktop/overview#requirements) page for steps you may have missed.

2. For information on the supported authentication methods, review the [authentication](https://docs.microsoft.com/azure/virtual-desktop/authentication) page.

3. For help setting up Azure multifactor authentication, review [how to enable Azure multifactor authentication](https://docs.microsoft.com/azure/virtual-desktop/set-up-mfa).
