# Outbound network connections from my app are failing

## **Recommended Documents**

Please review the following article to understand some of the common causes for Azure Functions apps to have issues with outbound connections.

* [Manage connections in Azure Functions](https://docs.microsoft.com/azure/azure-functions/manage-connections)

For .NET applications, Dependency Injection is a programmatic way of ensuring that outbound connections have a single client for the lifecycle of the Azure Functions app.

* [Use dependency injection in .NET Azure Functions](https://docs.microsoft.com/azure/azure-functions/functions-dotnet-dependency-injection)
