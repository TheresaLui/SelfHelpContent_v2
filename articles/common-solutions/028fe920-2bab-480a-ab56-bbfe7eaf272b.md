# Move Peering Resource across subscription

Subscription migration is not currently supported via API/Portal. To migrate resources from one subscription to another, it requires the resource under first subscription to be removed and then can be associated with the new subscription. 

## **Recommended Steps**
 
1. Please email it to [Peering Service](mailto:peeringservice@microsoft.com) with subject "Migrate subscription for resource **Old Subscription Id** **Resource Group Name** **Peering Name**". Please provide following details in the email:
	- Old Subscription Id
	- New Subscription Id
	- Resource Group Name
	- Peering Name
2. The process may take up to three days to complete. Microsoft will send notification when process is completed.
