# Azure DevOps Services buying build and release pipelines

Most users are able to resolve their issues purchasing Azure DevOps build and release pipelines using the steps below.

## **Recommended Steps**
* Before you create a support request, please check if there is an outage going on at [Azure DevOps Services Status](https://status.dev.azure.com)
* Want a quicker answer? For quick answers to common questions and issues, try the [Azure DevOps Virtual Agent](https://azuredevopsvirtualagent.azurewebsites.net/)


### **Buying Azure DevOps Cloud Service build and release pipelines (CI/CD)**
* Are you looking to buy Azure DevOps Services? Check out our pricing [here](https://azure.microsoft.com/pricing/details/devops/azure-devops-services/)
* Each Azure DevOps organization includes 1 Microsoft-hosted job with 1,800 minutes per month for CI/CD and 1 self-hosted job with unlimited minutes per month. To buy more build and release pipelines (CI/CD), you need to setup billing first before you make purchases.Please check [How to setup up billing](https://docs.microsoft.com/azure/devops/organizations/billing/set-up-billing-for-your-organization-vs?view=azure-devops)
* Prerequisites needed to setup billing and buy Azure DevOps user licenses are listed [here](https://docs.microsoft.com/azure/devops/organizations/billing/overview?view=azure-devops#prerequisites)
* To buy Microsoft-hosted CI/CD, check out [here](https://docs.microsoft.com/azure/devops/organizations/billing/buy-more-build-vs?view=azure-devops#buy-cicd)
* To buy self-hosted-hosted CI/CD, check out [here](https://docs.microsoft.com/azure/devops/organizations/billing/buy-more-build-vs?view=azure-devops#buy-cicd)
* You can buy more or reduce existing build and release pipelines (CI/CD) quantities in your [billing experience](https://docs.microsoft.com/azure/devops/organizations/billing/buy-more-build-vs?view=azure-devops#change-quantity-of-cicd)  


### **Buying Azure DevOps Server**
* If you are looking to buy Azure DevOps Server, check out [pricing and how to purchase here](https://azure.microsoft.com/pricing/details/devops/server/)
* For Azure DevOps Server, you can pay month-to-month through Azure or you can buy classic software licenses which requires a 3-year commitment. Buying through Azure provides bonus of entitling you to use our cloud service, so you can move to the cloud at your own pace.

### **Cloud Solution Providers (CSP)**
* Check out how Cloud Solution Providers can buy Azure DevOps build and release pipelines (CI/CD) [here](https://docs.microsoft.com/azure/devops/organizations/billing/csp/buy-csp-azure-devops?toc=%2Fazure%2Fdevops%2Forganizations%2Ftoc.json&bc=%2Fazure%2Fdevops%2Forganizations%2Fbreadcrumb%2Ftoc.json&view=azure-devops)
* If you are looking to change billing to a Cloud Solution Provider (CSP) Azure Subscription, check out the steps that both the customer and CSP has to perform [here](https://docs.microsoft.com/azure/devops/organizations/billing/csp/buy-csp-azure-devops?toc=%2Fazure%2Fdevops%2Forganizations%2Ftoc.json&bc=%2Fazure%2Fdevops%2Forganizations%2Fbreadcrumb%2Ftoc.json&view=azure-devops#change-billing-to-csp-azure-subscription)

### **Frequently Asked Questions**
* **Can I use the monthly Azure credits from my Visual Studio subscription to buy Azure DevOps build and release pipelines?**
No, you can't use the monthly Azure credits from [Visual Studio subscriptions](https://visualstudio.microsoft.com/products/subscriber-benefits-vs) to pay for Azure DevOps. Before you make purchases by using this type of Azure subscription, you must [remove your spending limit](https://azure.microsoft.com/pricing/spending-limits/).
* Checkout our extensive billing FAQs [here](https://docs.microsoft.com/azure/devops/organizations/billing/billing-faq?view=azure-devops)
