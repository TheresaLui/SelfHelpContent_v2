# Device Does not Boot up

Resolve issues with a device not starting using the following steps.


## **Recommended Steps**

1. Before you try to restart the device, take the shares offline. First, take the shares offline on the data server, and then do the same on the device.<br>
   This minimizes the possibility of data corruption.
2. In the local web UI, go to **Maintenance and Power** settings
3. Select **Shutdown** or **Restart**, depending on what you intend to do
