# Workbook isn't showing any data 


## **Recommended Steps**

If it is GPU workbook, the user need to have GPU enabled node-pool. More information in this [link](https://docs.microsoft.com/azure/azure-monitor/insights/container-insights-gpu-monitoring)
If it is NPM, the user need to have NPM enabled. More information in this [link](https://docs.microsoft.com/azure/virtual-network/kubernetes-network-policies?toc=/azure/azure-monitor/toc.json#monitor-and-visualize-network-configurations-with-azure-npm)
