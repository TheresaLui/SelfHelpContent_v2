# Missing metrics in Grafana chart 

Most customers can resolve issues with missing metrics in a Grafana chart using the following steps.

## **Recommended Steps**

* If you experience an issue connecting an Azure Monitor data source, [refer to the latest documentation](https://grafana.com/docs/grafana/latest/datasources/azuremonitor/)

* Learn about recent Grafana outages and issues from [GitHub](https://github.com/grafana/grafana/issues/) and from [the Grafana community](https://community.grafana.com/t/azure-log-analytics-issue-v6-6-0/25292)
