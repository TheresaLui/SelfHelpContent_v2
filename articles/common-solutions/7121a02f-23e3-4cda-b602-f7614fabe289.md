# Business to Consumer (B2C)

## **Recommended Steps**

### **Configure tokens in Azure Active Directory (AD) B2C** 

Learn how to configure the [lifetime and compatibility of a token](https://docs.microsoft.com/azure/active-directory-b2c/configure-tokens?pivots=b2c-custom-policy&WT.mc_id=Portal-Microsoft_Azure_Support).

### **Configure session behavior in AD B2C**

Learn how to configure the [session behavior](https://docs.microsoft.com/azure/active-directory-b2c/session-behavior?pivots=b2c-custom-policy&WT.mc_id=Portal-Microsoft_Azure_Support#configure-sign-out-behavior).

### **Can I ensure that only services that have proper credentials make calls to my REST API endpoint?**

Yes, learn how to [secure your REST API](https://docs.microsoft.com/azure/active-directory-b2c/secure-rest-api) with either HTTP basic, client certificate, or OAuth2 authentication.
