# VM Performance 

Azure Stack Edge VMs are basically Azure VMs and can be monitored in a similar way.

## **Recommended Steps**

1. Ensure you have deployed the VMs following the [steps provided](https://docs.microsoft.com/azure/databox-online/azure-stack-edge-gpu-deploy-virtual-machine-portal)

2. For instructions on enabling Azure Monitor, see [Azure Monitor on the VMs](https://docs.microsoft.com/azure/azure-monitor/insights/monitor-vm-azure#enable-azure-monitor-for-vms)

## **Recommended Documents**

* [To Enable Container Insights](https://docs.microsoft.com/azure/databox-online/azure-stack-edge-gpu-enable-azure-monitor#enable-container-insights)
