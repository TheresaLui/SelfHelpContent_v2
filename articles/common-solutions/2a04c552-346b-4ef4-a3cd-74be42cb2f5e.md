# Need help adding resources to DDoS protection plan 

## **Recommended Steps** 

Azure DDoS Protection Standard protects public IP addresses in virtual networks. If you have existing resources: 

1. Go to **Protected resources** 
2. Select **Add** 
3. Select the virtual network you want to add to your plan 

Alternatively, you can enable DDoS Protection Standard within the virtual network resource blade.

* [Enable DDoS Protection for a new or existing virtual network](https://docs.microsoft.com/azure/ddos-protection/manage-ddos-protection#enable-ddos-protection-for-a-virtual-network)
