# Need help configuring alerts 

Alerting can be configured for any of the Azure Monitor metrics exposed by DDoS Protection. In addition, through Azure Security Center, you get a unified view of alerts, including DDoS attack-related alerts and the actions taken to mitigate the attack in near-time. 

## **Recommended Documents**

* [Configure alerts through Azure Monitor](https://docs.microsoft.com/azure/ddos-protection/alerts#configure-alerts-through-azure-monitor)
* [Configure alerts through portal](https://docs.microsoft.com/azure/ddos-protection/alerts#configure-alerts-through-portal)
* [View alerts in Azure Security Center](https://docs.microsoft.com/azure/ddos-protection/alerts#view-alerts-in-azure-security-center)
