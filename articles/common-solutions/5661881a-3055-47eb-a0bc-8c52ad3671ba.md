# SQL Server Analysis Service Tabular Modeling 

Most users can resolve issues with SQL Server Analysis Services Tabular Modeling by using following steps.

## **Recommended Steps**

- Run the [SSASDiag Data Collection Tool](https://github.com/ssasdiag/SSASDiag/releases) to collect data on the Analysis Services server when the issue occurs. The resulting information can be used for cause analysis.

## **Recommended Documents**

- **How to design a tabular model**
	Review the available [tutorials](https://docs.microsoft.com/analysis-services/analysis-services-tutorials-ssas?view=asallproducts-allversions)
	In addition, you can determine the model mode, and learn tabular model design by reviewing the [tabular design overview](https://docs.microsoft.com/analysis-services/tabular-models/tabular-models-ssas?view=asallproducts-allversions).

- [Securing the Tabular Sematic Model with roles](https://docs.microsoft.com/analysis-services/tabular-models/roles-ssas-tabular?view=asallproducts-allversions&viewFallbackFrom=sql-server-ver15)

- [Troubleshooting errors in formulas](https://docs.microsoft.com/analysis-services/tabular-models/understanding-dax-in-tabular-models-ssas-tabular?view=asallproducts-allversions#troubleshooting-errors-in-formulas)
