# This is the title of the article. It is not displayed in the portal but is a required part of every article.

Most users are able to resolve their Data Sources' setup issues using the steps below.

## **Recommended Steps**

1. The person who is setting up the scan needs to be added to the Catalog resource permissions in the Azure portal as a Contributor for the scans to be allowed.

2. Verify that the account has the correct permissions to scan the data source in question. Please review the recommended documents, below to learn more.

3. Make sure that you are trying to connect to a supported data source. Please review the recommended documents, below to learn more.



## **Recommended Documents**

* Catalog Permissions: https://docs.microsoft.com/azure/purview/catalog-permissions
* Supported Data Sources: https://docs.microsoft.com/azure/purview/sources-and-scans
* Register Data Sources: https://docs.microsoft.com/azure/purview/bring-new-data-sources-under-management
