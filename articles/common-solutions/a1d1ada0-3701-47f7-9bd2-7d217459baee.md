# Azure data source setup issues

To resolve data source setup issues, use the following steps.

## **Recommended Steps**

1. The person setting up the scan needs to be added to the Catalog resource permissions in the Azure portal as a Contributor.

2. Verify that the account has the correct permissions to scan the data source in question. Review the following recommended documents.

3. Make sure that you are trying to connect to a supported data source. Review the following recommended documents.


## **Recommended Documents**

* [Catalog Permissions](https://docs.microsoft.com/azure/purview/catalog-permissions)
* [Supported Data Sources](https://docs.microsoft.com/azure/purview/sources-and-scans)
* [Register Data Sources](https://docs.microsoft.com/azure/purview/bring-new-data-sources-under-management)
