# Device Update import manifest self-help

It looks like you're having issues with creating an import manifest for your update content. Most users are able to resolve their import manifest issue using the steps below.

## **Recommended Steps**

1. Verify that your import manifest follows the appropriate JSON formatting.
2. Verify that your import manifest has values for **all** fields, as specified in the [import manifest documentation](https://docs.microsoft.com/iot-hub-device-update/update-manifest).
4. Check the [Device Update troubleshooting guide](https://docs.microsoft.com/iot-hub-device-update/troubleshoot-device-update) to see if your issue is addressed. If you are still encountering issues, please create a new support request.

* [Importing Update Content into Device Update](https://docs.microsoft.com/iot-hub-device-update/import-update)
* [Authoring a Device Update Import Manifest](https://docs.microsoft.com/iot-hub-device-update/update-manifest)
