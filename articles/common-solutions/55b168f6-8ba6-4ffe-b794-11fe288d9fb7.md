# Azure Sentinel - Threat Intelligence Data Connectors - Threat Intelligence Platforms (TIP)

## **Recommended Steps**

1.	If you never received data, or did receive data initially but has not since the initial connection :
	1.1. Open the Azure Portal and navigate to the Azure Sentinel service.   
	1.2 Choose the workspace where you have imported threat indicators using the TAXII Data Connector.  
	1.3 Navigate to the 'Data Connectors' section under the Configuration pane.  
	1.4 Open the 'Threat Intelligence Platforms (Preview)' data connector.  
	1.5 Check the last indicator received time on the Graph shown on the left pane of the connectors page. 
	1.6 Click on 'Go to Log Analytics' to see if there are indicators in Log Analytics  
		Use this query:  
			ThreatIntelligenceIndicator 
			| where SourceSystem == 'SecurityGraph'
