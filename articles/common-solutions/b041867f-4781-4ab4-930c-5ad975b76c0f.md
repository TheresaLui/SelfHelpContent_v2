# Managing an existing MHSM

Use these steps to manage your Azure hardware security module (HSM).

## **Recommended Steps**

* Control access to your managed HSM. Choose regions that support availability zones.
* Back up your HSM
* Turn on logging
* Turn on recovery options

For more information, see [Best practices for managing MHSM](https://docs.microsoft.com/azure/key-vault/managed-hsm/best-practices)
