# Device Update contact device self-help

It looks like you're having issues with contacting a device in a deployment. Most users are able to resolve their device contacting issue using the steps below.

## **Recommended Steps**

1. Verify that your device is connected to Device Update by checking if it shows up in the compliance view of Azure portal.
2. Check the [Device Update troubleshooting guide](https://docs.microsoft.com/azure/iot-hub-device-update/troubleshoot-device-update) to see if your issue is addressed. If you are still encountering issues, please create a new support request.

## **Recommended Documents**

* [Deploy an Update with Device Update](https://docs.microsoft.com/azure/iot-hub-device-update/deploy-update)
