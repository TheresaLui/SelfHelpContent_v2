# Issues creating a monitoring profile

SQL insights (preview) uses monitoring profiles to store your data collection settings. To create a monitoring profile, you need sufficient permissions.

## **Recommended Steps**

Verify that you have the [monitoring contributor role](https://docs.microsoft.com/azure/azure-monitor/roles-permissions-security#monitoring-contributor) in the subscription and the resource group where your monitoring profile will be placed.
