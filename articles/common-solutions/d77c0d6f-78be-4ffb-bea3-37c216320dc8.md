# Delete VMs
The Remove-AzureRmVM cmdlet removes a virtual machine from Azure.

## **Recommended Steps**
1.	Run the following cmdlet to remove a virtual machine from your device :PowerShellCopy Remove-AzureRmVM [-Name] <String> [-ResourceGroupName] <String>
2.	If you want some special options refer the link below for AzureRmVM mndlet
