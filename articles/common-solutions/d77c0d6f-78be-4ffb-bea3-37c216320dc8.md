# Delete VMs

The `Remove-AzureRmVM` cmdlet removes a virtual machine from Azure.

## **Recommended Steps**

1. Run the following cmdlet to remove a virtual machine from your device: 
```
PowerShellCopy Remove-AzureRmVM [-Name] <String> [-ResourceGroupName] <String>
```
2. For more options regarding the `AzureRmVM` mndlet, see the links in the following section
