# Azure Sentinel - Data collection - Connectors based on Logstash output plugin integration

## **Recommended Steps**

1. Make sure you have configured Logstash input plugin. You can validate Logstash is running successfully by using "service logstash status".
   Please check Logstash log file for any errors or warnings regarding Logstash activity - /var/log/logstash/logstash-plain.log .

2. Make sure your Microsoft Azure Sentinel Logstash output plugin version is between 7.0 and 7.9. Version can be found by using "/usr/share/logstash/bin/logstash --version".

3. Validate you are using Microsoft Azure Sentinel Logstash output plugin using "/usr/share/logstash/bin/logstash-plugin list | grep microsoft-logstash-output-azure-loganalytics".
