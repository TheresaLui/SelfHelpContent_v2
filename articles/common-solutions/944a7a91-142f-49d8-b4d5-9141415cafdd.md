# Device Return
You can place a Return request for your Azure Stack Edge device from the Overview blade of your resource within Azure portal.

## **Recommended Steps**
 
1. Erase data from the device
2. In the local web UI, go to Maintenance > Device reset and select *Reset device*
3. Initiate device return in Azure portal
4. Go to your Azure Stack Edge resource in the Azure portal. In the Overview, go to the command bar in the right pane and select Return device.
5. Fill up the necessary details and Initiate Return. This will result in a mail being sent to Azure Stack Edge operations team with your request.
6. Operations team will get back to you with return shipment label and instructions to schedule a pickup
