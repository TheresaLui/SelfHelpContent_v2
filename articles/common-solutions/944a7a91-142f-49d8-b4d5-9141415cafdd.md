# Device Return

Use the following instructions to place a return request for your Azure Stack Edge device.

## **Recommended Steps**
 
1. Erase data from the device
2. In the local web UI, go to **Maintenance** > **Device reset** and select **Reset device**
3. From the Azure Stack Edge resource in the Azure portal, go to **Overview**, and in the command bar in the right pane, select **Return device**
5. Fill in the necessary details and select **Initiate Return**.<br>
   An email will be sent to Azure Stack Edge operations team with your request.
   The Operations team will send you a return shipment label and instructions to schedule a pickup.
 
 
## **Recommended Documents**
 
* [Pack the device and schedule a pickup with the carrier following the steps](https://docs.microsoft.com/azure/databox-online/azure-stack-edge-return-device)
