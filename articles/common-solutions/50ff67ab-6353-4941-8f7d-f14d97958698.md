# Kubernetes How To

On your Azure Stack Edge device, a Kubernetes cluster is created when you configure a compute role. After the Kubernetes cluster is created, you can deploy containerized applications on the Kubernetes cluster in Pods. To manage and connect to the cluster locally from a client machine, use a native tool, such as kubectl. 

For steps on how to work with your Kubernetes cluster, including how to troubleshoot common issues (such as changing the range of the internal IP address) review the following steps.
