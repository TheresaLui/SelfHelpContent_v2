# How do I handle studio access issues and connection error?

## **Recommended Steps**

We had trouble reaching the server. This could be due to a few reasons:
* Your network or internet access is down
* You are using a firewall which is blocking access
* A browser extension such as an adblocker is mistakenly blocking access

Click Refresh now to try connecting again.
