#  ADF limits and how to increase them

Azure Data Factory is a multi-tenant service that has the following default limits in place to make sure that customer subscriptions are protected from each other's workloads. For increasing ADF limits, customers need to contact support.

## **Recommended Steps**

1. Review [ADF limits](https://github.com/MicrosoftDocs/azure-docs/blob/master/includes/azure-data-factory-limits.md)
2. If you need to increase the limits despite optimizations, contact support
