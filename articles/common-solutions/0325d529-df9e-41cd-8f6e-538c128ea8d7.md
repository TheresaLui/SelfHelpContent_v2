# Issues signing into Microsoft applications

## **Recommended Steps**

Resolve issues signing in to Microsoft applications by using the following information.

**I'm having issues signing in to a Microsoft application**

* [I can’t sign in to a Microsoft application](https://docs.microsoft.com/azure/active-directory/manage-apps/application-sign-in-problem-first-party-microsoft?WT.mc_id=UI_AAD_Apps_Sign_In_Support_L2_MicrosoftApp)

**I'm having other issues while signing in to an application**

* [I can't log in using the service principal](https://docs.microsoft.com/cli/azure/create-an-azure-service-principal-azure-cli#manage-service-principal-roles)
