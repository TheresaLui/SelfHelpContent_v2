# Authentication Errors

## **Recommended Steps**

* Check any authentication related errors from [Activity troubleshooting guidance](https://docs.microsoft.com/azure/data-factory/data-factory-troubleshoot-guide) 

* If any authentication issue happening from any connectors, please refer to [connector troubleshooting guide]( https://docs.microsoft.com/azure/data-factory/connector-troubleshoot-guide). 

* If you see any authentication issue via Managed Vnet and Private link, please refer to [security and access control troubleshoot guide](https://docs.microsoft.com/azure/data-factory/security-and-access-control-troubleshoot-guide) 

* If you see any authentication issue via Self-Hosted IR, please refer to [self hosted Integration runtime troubleshoot guide](https://docs.microsoft.com/azure/data-factory/self-hosted-integration-runtime-troubleshoot-guide)
