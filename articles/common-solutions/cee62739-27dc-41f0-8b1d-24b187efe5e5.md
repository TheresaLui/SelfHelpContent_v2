# EdgeArm credentials

The procedure to set the password can be different depending upon whether you use the Azure portal or the PowerShell cmdlets.

## **Recommended Steps**

1.	In the Azure Portal, go to the Azure Stack Edge resource that you created to manage your device. Go to **Edge compute** > **Get started**.
2.	In the right pane, from the command bar, select **Reset Edge ARM password**.
