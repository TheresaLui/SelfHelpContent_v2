# EdgeArm credentials

The procedure to set the password differs if you use the Azure portal versus the PowerShell cmdlets.

## **Recommended Steps**

1.	In the Azure portal, go to the Azure Stack Edge resource that you created to manage your device. 
2.	Go to **Edge compute** > **Get started**.
3.	In the right pane, from the command bar, select **Reset Edge ARM password**.

* [Detailed steps for resetting password](https://docs.microsoft.com/azure/databox-online/azure-stack-edge-j-series-set-azure-resource-manager-password)
