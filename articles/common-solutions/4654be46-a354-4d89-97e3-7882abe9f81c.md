# Job started to fail with error

## **Recommended Steps**

1. Check if there are outages in the region where this job is running on [Azure status page](https://status.azure.com/status)<br>
2. Understand the error that caused the job to fail by using the Job Browser in Visual Studio or under the Job management tab for the ADLA account in the Azure portal<br>
