# Device Update resource creation self-help

It looks like you're having issues with creating Device Update resources. Most users are able to resolve their resource creation issue using the steps below.

## **Recommended Steps**

1. Please refer to the Device Update documentation for [creating Device Update resources](https://docs.microsoft.com/azure/iot-hub-device-update/create-device-update-account) and confirm that you have followed the correct steps for creating a Device Update instance.
2. Check the [Device Update troubleshooting guide](https://docs.microsoft.com/azure/iot-hub-device-update/troubleshoot-device-update) to see if your issue is addressed. If you are still encountering issues, please create a new support request.
