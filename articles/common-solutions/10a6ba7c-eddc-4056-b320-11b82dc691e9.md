# SQL Server Analysis Service Multidimensional Modeling
Most customers can resolve issues  with  SQL Server Analysis Service Multidimensional Modeling by using following steps.

## **Recommended Steps**

- Please run the [SSASDiag Data Collection Tool](https://github.com/ssasdiag/SSASDiag/releases) to collect data on the Analysis Services server when problem occurs. The info can be used for cause analysis.

## **Recommended Documents**

- **How to design Multidimensional model**
   You can review the tutorials available [here](https://docs.microsoft.com/analysis-services/analysis-services-tutorials-ssas?view=asallproducts-allversions).In addition,  you can determine the model mode, and learn Tabular Model designer by reviewing the [Multidimensional Designing overview](https://docs.microsoft.com/analysis-services/multidimensional-models/multidimensional-models-ssas?view=asallproducts-allversions)

- [Securing the Multidimensional Model with Roles](https://docs.microsoft.com/analysis-services/multidimensional-models/roles-and-permissions-analysis-services?view=asallproducts-allversion)
