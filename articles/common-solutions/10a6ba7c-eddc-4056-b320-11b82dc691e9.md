# SQL Server Analysis Service Multidimensional Modeling

This article can help most customers resolve issues with SQL Server Analysis Service (SSAS) Multidimensional Modeling.

## **Recommended Steps**

-  Run the [SSASDiag Data Collection Tool](https://github.com/ssasdiag/SSASDiag/releases) to collect data on the Analysis Services server for cause analysis.

## **Recommended Documents**

- **How to design a Multidimensional model**<br>
   [Review these tutorials](https://docs.microsoft.com/analysis-services/analysis-services-tutorials-ssas?view=asallproducts-allversions). To determine the model mode and learn Tabular Model designer, review the [Multidimensional Designing overview](https://docs.microsoft.com/analysis-services/multidimensional-models/multidimensional-models-ssas?view=asallproducts-allversions).

- [Secure the Multidimensional Model with Roles](https://docs.microsoft.com/analysis-services/multidimensional-models/roles-and-permissions-analysis-services?view=asallproducts-allversion)
