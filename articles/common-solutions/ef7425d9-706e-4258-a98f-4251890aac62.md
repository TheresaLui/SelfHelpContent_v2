# Business to Consumer (B2C)

## **Recommended Steps**

### **How can I monitor my Azure AD B2C?**  

You can use [Azure Monitor to monitor Azure Active Directory B2C](https://docs.microsoft.com/azure/active-directory-b2c/azure-monitor) sign-in and auditing logs to different monitoring solutions. You can retain the logs for long-term use or integrate with third-party security information and event management (SIEM) tools to gain insights into your environment.

### **Can I track user behavior in Azure Active Directory B2C?**  

Yes, Azure Active Directory B2C (Azure AD B2C) supports sending event data directly to Application Insights by using the instrumentation key provided to Azure AD B2C. With an Application Insights technical profile, you can get detailed and customized event logs for your user journeys. Get started to [track user behavior in Azure Active Directory B2C using Application Insights](https://docs.microsoft.com/azure/active-directory-b2c/analytics-with-application-insights).
