# Device Update device status self-help

It looks like you're having issues with viewing the status of a device in a deployment. Most users are able to resolve their device status issue using the steps below.

## **Recommended Steps**

1. Verify that your device is connected to Device Update by checking if it shows up in the compliance view of Azure portal.
2. Check the [Device Update troubleshooting guide](https://) to see if your issue is addressed. If you are still encountering issues, please create a new support request.
