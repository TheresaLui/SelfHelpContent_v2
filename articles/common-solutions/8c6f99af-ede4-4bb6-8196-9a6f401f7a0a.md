# IOT Edge Compute

To configure compute on your Azure Stack Edge, you'll create an IoT Hub resource.

## **Recommended Steps**

1. On the Configure Edge compute tile, select Configure compute
2. On the Configure Edge compute blade, input the IOT Hub Details
3. Choose from New or Existing
4. By default, a Standard tier (S1) is used to create an IoT resource. To use a free tier IoT resource, create one and then select the existing resource.
5. In each case, the IoT Hub resource uses the same subscription and resource group that is used by the Azure Stack Edge resource
