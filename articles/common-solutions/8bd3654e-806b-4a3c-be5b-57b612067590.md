# Azure Sentinel - Analytic rules  - Entities in scheduled  alert rule

Resolve issue related to entities in scheduled alert rules in the following steps.

## **Recommended Steps**

1.	Make sure that the entities are mapped on the rule

2.	Validate the entity mapping configuration according to [analytics rules documentation](https://docs.microsoft.com/azure/sentinel/tutorial-detect-threats-custom).
