# Device Update incorrect targeting self-help

It looks like you're having issues with targeting a Device Update group for a deployment. Most users are able to resolve their targeting issue using the steps below.

## **Recommended Steps**

1. Verify that you correctly defined the device group.
2. Verify that you have correctly selected the desired group when creating the deployment.
3. Verify that the deployment's update content is applicable to the devices in your group.
4. Check the [Device Update troubleshooting guide](https://docs.microsoft.com/iot-hub-device-update/troubleshoot-device-update) to see if your issue is addressed. If you are still encountering issues, please create a new support request.

* [Creating a Device Update group](https://docs.microsoft.com/iot-hub-device-update/create-update-group)
* [Deploy an Update using Device Update](https://docs.microsoft.com/iot-hub-device-update/deploy-update)
