# Policies

You can use Azure Policy to create and assign policies for implementing governance for resource consistency, regulatory compliance, security, cost, and management. 

## **Recommended Steps**

### **Built-in policies**

Azure ML includes a list of [built-in policies](https://docs.microsoft.com/azure/machine-learning/policy-reference#built-in-policy-definitions) that can be assigned through Azure Policy. For instructions on assigning policies, see the [documentation](https://docs.microsoft.com/azure/governance/policy/assign-policy-portal).

### **Custom policies**

If the built-in policies don't work, you can create custom policy definitions through Azure Policy. For instructions on creating customer policies, see the [documentation](https://docs.microsoft.com/azure/governance/policy/tutorials/create-custom-policy-definition).
