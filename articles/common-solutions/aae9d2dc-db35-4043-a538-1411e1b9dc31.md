# Connect to Azure

Check if the necessary ports as detailed in the support article are opened on your firewall in order for your Azure Stack Edge device to communicate with Azure. 

## **Recommended Steps**

1. The device (source) IPs should always be set to all the cloud-enabled network interfaces
2. The destination IPs should be set to Azure datacenter IP ranges
