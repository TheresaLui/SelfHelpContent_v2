# Adaptative Network Hardening

## **Recommended Documents**

Applying network security groups (NSG) to filter traffic to and from resources improves your network security posture.

Learn how to improve the security posture by hardening the NSG rules based on the actual traffic patterns in Security Center, see [Adaptive network hardening in Azure Security Center](https://docs.microsoft.com/azure/security-center/security-center-adaptive-network-hardening).
