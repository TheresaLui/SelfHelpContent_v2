To be able to use Regulatory Compliance, Azure Security Center needs to be on standard pricing tier on the subscription level.

	• In the case that the Regulatory Compliance UI is not loading correctly, please try :

		1. Clear browser cache 
		2. Use a different browser 
		3. Try to load it from different network location

	• The compliance dashboard surfaces security assessments and recommendations as they align to specific compliance requirements - so if some recommendation result is not accurate, please refer to the recommendation section for further analysis.
