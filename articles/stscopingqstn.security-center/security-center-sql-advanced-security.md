# SQL Advanced Data Security

## **Recommended Documents**

Learn how to set up SQL Advanced Data Security in Azure Defender to provide SQL Vulnerability Assessment and Advanced Thread Protection, see [Introduction to Azure Defender for SQL](https://docs.microsoft.com/azure/security-center/defender-for-sql-introduction).

Enable Azure Defender for SQL once to enable all these included features. With one click, you can enable Azure Defender for all databases on your server in Azure or in your SQL Managed Instance.
