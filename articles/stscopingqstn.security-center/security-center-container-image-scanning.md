# Container via Image Scanning

## **Recommended Documents**

Learn how to use the built-in vulnerability scanner to scan the container images stored in your Azure Resource Manager-based Azure Container Registry, see [Use Azure Defender for container registries to scan your images for vulnerabilities](https://docs.microsoft.com/azure/security-center/defender-for-container-registries-usage).

When **Azure Defender for container registries** is enabled, any image you push to your registry will be scanned immediately.
