# Error UserErrorRpCollectionDeleted

<!--issueDescription-->
 We have identified your backup operation failed because the Restore Point collection used during Backup has been deleted.
<!--/issueDescription-->

## **Recommended Steps**
To resolve this issue retry the operation and do not delete any Restore Point collection created under the Resource Group.
