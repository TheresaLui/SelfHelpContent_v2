# Error UserErrorBcmDatasourceNotPresent 	

<!--issueDescription-->
We have identified that your backup operation failed because the previous backup operation was paused.
<!--/issueDescription-->

## **Recommended Steps**
To resolve this issue, perform the below steps:

* Ensure the VM is backed up
* Check if the previous backup operation is paused. If yes, then resume the paused backup operation.
