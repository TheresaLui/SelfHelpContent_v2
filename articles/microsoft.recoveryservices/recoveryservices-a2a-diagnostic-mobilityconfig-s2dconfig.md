# Mobility agent configuration fails because the storage spaces direct is unsupported

<!--issueDescription-->
Mobility agent configuration on host machine failed with error **A2AMobilityServiceConfiguratorUnknownException (ID: 151075)**. This happens because the storage spaces direct configuration has limited support.
<!--/issueDescription-->

## **Recommended Steps**

Configure the storage spaces direct until it is supported.
