# Check if both VMs running?

Make sure that both VMs are started and in Succeded state

## Recommended Steps

1. Navigate to the destination VM in ASC Resource Explorer
2. Confirim that the Current Power State is "running" and State is "Converged"
3. Navigate to the source VM in ASC Resource Explorer
2. Confirim that the Current Power State is "running" and State is "Converged"
