# Gather necessary data

Before starting the troubleshooting process, make sure that you have gathered all necessary data.

## Recommended Steps

1. Gather the source and destination (private) IP addresses and ports, as well as the names of the VMs
2. Make sure that the IP addresses specified by the customer match the VMs that they have mentioned
