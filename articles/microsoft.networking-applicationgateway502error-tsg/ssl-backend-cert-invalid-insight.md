# Application Gateway 502 Error: Certificate Invalid Insight

<!--issueDescription-->
Application Gateway verifies the validity of the backend server certificate. If it is expired, the server would be considered unhealthy. To resolve this, renew the certificate and update the server configuration to use the new and valid certificate.
<!--/issueDescription-->
