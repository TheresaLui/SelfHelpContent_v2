# Partner Center Other Portals Data privacy Data privacy inquiry

Microsoft must comply with data subject rights requests for access, deletion, requests to stop processing or any other privacy related question regardless of where the data resides.

## **Recommended Steps**

* Raise your privacy question or concern to our privacy team [here](https://privacy.microsoft.com/privacy-questions)

* If you are a global administrator for your account, request data deletion in Azure [here](https://azure.microsoft.com/support/create-ticket/)

* If you want to request deletion of support request data, please reopen the request if needed and then add a note [here](https://partner.microsoft.com/dashboard/support/servicerequests)
