# API failures when trying to create, view or update Co-sell opportunities

## **Recommended Steps**

* If you are receiving a 401 error, please verify your authorization token
* If you are receiving a 403 error, please verify you have the role of *‘Referrals admin’*. You can view your assigned role in your *‘Partner Center profile (on the top right corner) > My profile > Account > Partner Center permissions -> ‘Click here to view permissions’*.
* If you are not receiving any results, please verify your referral ID and the filter parameters

**Note**: Please include the endpoint URL, request payload, the response code, the error message received, the [network trace](https://docs.microsoft.com/azure/azure-portal/capture-browser-trace), the MPN ID & the referral ID when filing a support ticket
