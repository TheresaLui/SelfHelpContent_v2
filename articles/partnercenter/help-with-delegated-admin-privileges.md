# Help with delegated admin privileges

## **Recommended Steps**

Before submitting a support ticket using the below recommended documentation as a reference:

- Ensure you have a **reseller relationship** with your customer
- For Azure, ensure you have the propper **role access** on your customer's Azure subscription
