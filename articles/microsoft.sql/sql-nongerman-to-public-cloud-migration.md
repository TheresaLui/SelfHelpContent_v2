# Migration from Microsoft Cloud Germany to Global Azure cloud ONLY.      

## **Recommended Steps**
The problem type  selected (*Migration from Microsoft Cloud Germany*) is applicable ONLY for migrating database from German Cloud Subscription to Public Cloud and should be used from your Germany Subscription.  
To migrate your database in Germany Cloud  into Azure Public cloud environment, please use your Azure Germany Subscription via Azure Germany Portal and select this problem subtype to open new service request.

For moving your resources from Non-German Subscription please use the Problem type Backup & Restore Or Migrating to Azure.
