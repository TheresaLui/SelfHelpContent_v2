# Migration from the Azure Germany cloud to Azure Global cloud ONLY.      

## **Recommended Steps**

The problem type  selected (*Migration from Azure Germany cloud*) is applicable ONLY for migrating a database from the Azure Germany cloud subscription to the Public Cloud and should be used from your Germany subscription.  
To migrate your database from the Azure Germany cloud into the Azure Public cloud environment, use your Azure Germany subscription via the Azure Germany Portal and select this problem subtype to open new service request.

To move your resources from a non German subscription, for **Problem type**, use **Backup & Restore** or **Migrating to Azure**.
