# How to questions: Login and Users

## **Recommended Steps**

### How to questions: Login and Users

* Azure SQL DB supports two types of logins - SQL Server login and Azure Active Directory (AAD) login. To create Azure AD logins, you must set up an AD administrator.
For handling the issues related to creating Login / User accounts for SQL Authentication or AAD, please visit [Authorize database access](https://docs.microsoft.com/azure/azure-sql/database/logins-create-manage?WT.mc_id=pid:13491:sid:32745433/). <br>
