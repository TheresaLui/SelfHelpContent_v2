# Issue Category

### **Check Connection Endpoint**
Note that when using Azure Cosmos DB's API for MongoDB accounts, the 3.6 version of accounts have the endpoint in the format `*.mongo.cosmos.azure.com` whereas the 3.2 version of accounts have the endpoint in the format `*.documents.azure.com`.  


Customers may face variety of issues with MongoDB Connectivity and Availability such as:

1. Client connection errors and socket timeout
2. ExceededTimeLimit Error 
3. TooManyRequests
4. MongoDB wire version issues
5. ExceededMemoryLimit
6. Index path corresponding to the specified order-by item is excluded
7. MongoDB wire version issues
8. Mongo Shell Not Working
9. Connection Debug
10. Connectivity Issues
11. Connectivity Issues with Mongo API Geo-Replicated account
12. Socket Connection Exceptions
13. MongoSocketWriteException Errors
