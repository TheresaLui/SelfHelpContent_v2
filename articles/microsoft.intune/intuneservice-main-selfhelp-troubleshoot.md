# Where can I troubleshoot issues with Intune?

## **Recommended steps**

Start troubleshooting user issues from the [Troubleshoot blade](data-blade:Microsoft_Intune_DeviceSettings.TroubleshootBlade). After loading a user click the Assignments dropdown to switch troubleshooting topics.
