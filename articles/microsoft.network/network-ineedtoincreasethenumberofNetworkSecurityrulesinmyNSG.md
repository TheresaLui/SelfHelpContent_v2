# I need to increase the number of Network Security rules in my NSG.

## **Recommended steps**
Default limit on the number of NSG rules is 200, by raising a support ticket this can be increased up to 400 (Classic) and 500 (Azure Resource Manager). If you need to increase the number of security rules in your NSG follow the steps below.<br>

1. Create New Support Request<br>
[New Support Request](Show blade)
2. Select the Issue type as Quota.<br>
3. Select the Quota type as Networking: Classic (or Networking: ARM) and enter details of the NSG in details section.<br>
