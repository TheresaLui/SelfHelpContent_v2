# Issues with on-premise connectivity

## **Recommended steps**

To resolve the most common issues, try one or more of the following methods.

1. Check [connection configuration](data-blade:Microsoft_Azure_Network.ConnectionInfoBlade) is setup correctly.
2. Set up and run the connectivity debug tool to identify & correct issues. <br>[Run the connectivity tool](https://github.com/Azure/NetworkMonitoring/tree/master/AzureCT#azure-connectivity-toolkit-azurect)
