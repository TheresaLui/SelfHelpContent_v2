# TS4 SC7 Detected throttling due to high charges for query operations operation type 15

<br>
#### **Detected rate limiting/throttling(429) due to high charges for query operations (operation type 15) that trigger scan and large no of collisions.**

<br>Step 1. Download the ETL files for the timeperiod and filter for the EndRequests for the documentserviceid and operations.

<br>Step 2. Search for the max_MaxRequestCharge in the ETL file, get activity id and look at the query execution stats to see how many documents got retrieved and what the hit count was

<br> Step 3. To continue troubleshooting this issue, please select and follow **Solution 8** from start menu  to suggest the customer trying the following 3 workaround.

<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**Solution 8 Increase retries/wait time or RU Limit or reduce required RU consumption**

* Increase retries/wait timer
* Increase RU Limit
* Reduce required RU consumption
