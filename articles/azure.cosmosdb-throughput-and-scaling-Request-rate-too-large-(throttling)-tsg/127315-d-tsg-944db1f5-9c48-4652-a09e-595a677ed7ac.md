# DTS0DC2 Select preferred tool for 429 more than 30d

<br> Please select your preferred troubleshooting tool.
<br> Be sure that you're only able to select the investigation tools which the data retention period is longer than the days that the customer experienced rate limiting/throttling(429).
<br>
<pre>
Investigation Tool                                 Retention Period
Azure Cosmos DB Portal or Azure Monitor Logs       < 7 days 
Azure Monitor Metrics                              < 30 days
ASC Resource Explorer or Jarvis                    >= 30 days
</pre>
