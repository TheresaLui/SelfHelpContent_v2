# TS0 DC3 Check if 429 experienced with kusto query

<br>Run following Kusto query with proper database account name and the time range to check if we experienced rate limiting/throttling(429).

<pre>
let accountName = {globalDatabaseAccountName};
let startTime = datetime(2020-11-26 00:00:00);
let endTime = datetime(2020-11-27 23:00:00);
BackendEndRequest5M
| where GlobalDatabaseAccountName == accountName
| where TIMESTAMP > startTime and TIMESTAMP < endTime
| where StatusCode ==429
</pre>
