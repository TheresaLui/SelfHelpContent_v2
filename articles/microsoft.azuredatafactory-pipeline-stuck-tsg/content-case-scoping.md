# Case Scoping

Many times we see customers experiencing issues with the execution of the pipeline. Either the pipeline execution is delayed or queued. The issues can be due to an existing run or concurrency setting configured by the customer. 

**NOTE:** Customer can achieve concurrency at the pipeline level or multiple trigger runs can also cause pipelines to run concurrently.
