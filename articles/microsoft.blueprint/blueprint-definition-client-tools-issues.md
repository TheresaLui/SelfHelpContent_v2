# Azure Blueprints - Problems with client tools

A blueprint can be assigned through the following clients:
* Powershell
* SDK for .NET
* REST API

## **Recommended Steps**

Go through the blueprint documentation for the client tool you are using.
 * [Powershell](https://docs.microsoft.com/powershell/module/az.blueprint/?view=azps-4.1.0#blueprint).
 * [SDK for .NET](https://docs.microsoft.com/dotnet/api/overview/azure/blueprints/management?view=azure-dotnet)
 * [REST API](https://docs.microsoft.com/rest/api/blueprints/)

 Go through the troubleshooting guidance for blueprint errors.
 * [Troubleshooting blueprints](https://docs.microsoft.com/azure/governance/blueprints/troubleshoot/general)
