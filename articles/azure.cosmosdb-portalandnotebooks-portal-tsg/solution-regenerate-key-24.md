# Regenerate Key

<!--issueDescription-->

Dear customer,

If you are unable to regenerate the key for DocumentDB account using Mongo API, this could happen due to a portal bug or a portal issue.

**Solution**  
 If you  runs into such issues, while the portal issue is being looked at and addresses, we can suggest rotating the keys of the account programmatically. Please see Regenerate Account Key in the following document:
- https://docs.microsoft.com/en-us/azure/documentdb/documentdb-manage-account-with-powershell?

Thank you.
