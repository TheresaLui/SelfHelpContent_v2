# Run the subscription enablement diagnostic

There is a diagnostic that can be run quickly to check if the customer subscription qualifies to be enabled. The diagnostic output will give instructions on what to do next.

### Recommended Steps
1. Within Azure Support Center, select Resource Explorer at the top of the screen.
2. Select the subscription id GUID in the left pane at the top
3. Select 'Diagnostics' option in the middle of the screen
4. Select the "Enable Port 25" button to run the diagnostic
5. Choose the option below that is closest to what was displayed
