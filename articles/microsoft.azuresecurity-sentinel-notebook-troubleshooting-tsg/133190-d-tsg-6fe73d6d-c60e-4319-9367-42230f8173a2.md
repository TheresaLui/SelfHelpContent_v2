# Kernel Not Found

Kernel Not Found

Notebooks will store the last used Kernel name as meta-data with the notebook, if you then load the notebook in an environment without that kernel name you will get a Kernel not found error. 

How to resolve:
Select a kernel of the correct language and version and click ‘Set Kernel’
