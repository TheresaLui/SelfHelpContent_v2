# When you want to remove the Public IP on the compute cluster or instance

This helps when users want to remove the Public IP that gets created for a training cluster or a compute instance inside the users network 

## **Recommended Steps**

1. Unfortunately, there is no way to remove the public IP on these resources currently. The product team is working on removing this limitation in the future.
