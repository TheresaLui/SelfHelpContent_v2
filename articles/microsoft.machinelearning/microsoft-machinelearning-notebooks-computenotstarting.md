# Compute not starting

Your compute instance may have issues. Most users are able to resolve their Compute Instance issue by using the follwing steps.

## **Recommended Steps**
1. Click "Compute" in the Azure ML Studio menu bar
2. On the compute page locate the Compute Instance with the issues
3. Try to restart the Compute Instance from here
4. If the Compute Instance continues to fail, make sure the compute is stopped and create a new one
