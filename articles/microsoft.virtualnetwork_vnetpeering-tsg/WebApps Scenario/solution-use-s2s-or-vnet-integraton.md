# Using WebApp over VNet Peering
<!--issueDescription-->

To allow a Webapp to communicate over a VNet Peering connection we have two options. The first is to use the webapp feature "VNet Integration" or the second option is to create a Vnet-to-Vnet VPN Gateway connection between the two VNets. The links to these solutions are below.
