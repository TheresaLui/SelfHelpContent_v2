# Azure Media Services analyzing with Indexer

Azure Media Indexer is a Media Processor available in the V2 (legacy) API that you use to generate closed captioning tracks and keywords.

Media Services also has a [new set of presets in the v3 API](https://docs.microsoft.com/azure/media-services/latest/analyzing-video-audio-files-concept) called Audio and Video Analyzer that replace the legacy "Indexer v1 and v2 Preview". 

### Should I use Indexer v1, Indexer v2 Preview, or the new Audio and Video Analyzer?

It is recommmended to move to the [v3 API Audio and Video Analyzer](https://docs.microsoft.com/azure/media-services/latest/analyzing-video-audio-files-concept)and use these newer presets which have more advanced capabilities and performance. 

Also, you may consider consuming the [Video Indexer APIs](http://video.ai) if you need more advanced video AI capabilities and an integrated portal and search experience.
