# Azure Media Services Explorer tool

### **Azure Media Services Explorer tool**

The Media Services Explorer (AMSE) tool is an open source community project to build a simple Windows desktop application that demonstrates the usage of the Media Services APIs.  The tool is not supported officially by the Media Services product team. 

* [Blog: Managing Media workflows with the AMSE tool](http://azure.microsoft.com/blog/2014/10/08/managing-media-workflows-with-the-new-azure-media-services-explorer-tool)

Azure Media Services Explorer (AMSE) is a Winforms/C# application for Windows that does upload, download, encode and stream VOD and live content with Azure Media Services.

Issues related to the AMSE tool should be submitted to the open source repository [Issue List](https://github.com/Azure/Azure-Media-Services-Explorer/issues).

### **How do I submit a bug on the tool?**

* [Open a bug in the issues list](https://github.com/Azure/Azure-Media-Services-Explorer/issues/new)

### **Can I contribute back to the tool or suggest bug fixes?**

The AMSE project has adopted the Microsoft Open Source Code of Conduct. For more information see the Code of Conduct FAQ or contact opencode@microsoft.com with any additional questions or comments.

Contact: [amse@microsoft.com](mailto://amse@microsoft.com)
