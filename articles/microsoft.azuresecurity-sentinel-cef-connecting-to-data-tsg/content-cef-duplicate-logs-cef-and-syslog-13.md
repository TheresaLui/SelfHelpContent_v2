# Duplicate logs - syslog & CEF

Customer is using the same machine to forward both plain Syslog and CEF messages. In order to use the same machine to forward both plain Syslog and CEF messages, please follow this doc  instructions [Configure the log analytics agent](https://docs.microsoft.com/azure/sentinel/connect-syslog#configure-the-log-analytics-agent)
